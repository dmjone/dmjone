Act I · The Belief

Education is a fundamental human right.

A first-year computer science student in a small town deserves the same materials as one in a metro. The same compilers. The same tutorials. The same chance to publish their first package. The platform exists because nobody else had built that, end to end, in the open.

"We bridge the education gap by giving free, high-quality computer science resources to anyone who wants them. The fact that you cannot afford a course, a book, or a paid editor must not be the reason your engineering career stops." · The dmj.one mission, restated here

Act II · The Pillars

Four commitments, kept on every page.

Every product here inherits the same constraints from this platform. They are not aspirations. They are the floor.

Pillar i.

Welfare first.

Every project ships free at point of use. Income comes from work for others, not from people learning here.

Pillar ii.

Education first.

If a tutorial helps one student finish their first program, it stays online for every student that follows.

Pillar iii.

India first.

Designed for the slow phone, the patchy network and the small town. If it works there, it works everywhere.

Pillar iv.

Accessibility first.

WCAG 2.2 AAA from line one. Screen reader, keyboard, low-vision, low-bandwidth, all on the same site, with no second-class path.

Act IV · The Archive

Public. Auditable. Built to outlast.

The platform is a static site, deployed redundantly, backed by audited code and named contributors. The repository moved from `dmjone/dmjone.github.io` to `dmjone/dmjone`: the older URL is preserved as a redirect, no link ever broke for a learner.

Hosting · redundant

GitHub Pages and Cloudflare Pages.

Deployed automatically on every push to both providers. If one cache misses, the other still serves. Zero-downtime publishing for free.

Pipeline · audited

CodeQL, dependency review, security scans.

GitHub Actions run code-quality and security gates before any deploy. A static site, but never an unaudited one.

Licence · AGPL v3

Free in code. Free in writing. Free for the next student.

The educational content is AGPL v3. Anyone forking the platform must keep the next student's access just as free. The licence is the contract that keeps the mission honest. (Individual products here ship under their own licences · most are Apache 2.0 with patent retaliation.)
