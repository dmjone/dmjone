Manifesto.

Most software pretends India is a startup conference. It isn't. It's a 1.4 billion person, twenty-two language, low-bandwidth, slow-phone, small-town country where the next user isn't on Wi-Fi and didn't grow up reading English error messages.

I build for that India. Not because it's noble, but because if it works in India, it works everywhere. Every product here ships with the same defaults: works offline, speaks your language, loads in under three seconds on a ₹6,000 phone, and refuses to leak your data even to me.

I don't write code to add features. I write code to fix things that hurt real people · a tanker that never came, an ASHA worker without diagnostics, a farmer whose price never reaches the mandi, a student whose exam was already leaked.

Fifty-nine of those fixes are here. The next one is yours.

· D.M. · Solan, Himachal Pradesh · May 2026
