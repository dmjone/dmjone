Introduction to Computer Graphics

2025, December 14

1. Introduction to Computer Graphics

Computer Graphics is the discipline that converts abstract data and mathematical descriptions into visual representations that humans can perceive, interpret, and interact with. At its core, it answers one question: how can numbers be transformed into meaningful images on a screen. Every pixel displayed is the result of precise computations involving geometry, color, light, and perception.

The Goal

Q: At its core, what is the single main task of Computer Graphics?

A: Transforming abstract data/numbers into meaningful visual representations.

1.1 Definition of Computer Graphics

Computer Graphics is the branch of computer science concerned with the creation, storage, manipulation, and display of visual information using computers. It transforms numerical data (coordinates, colors, intensities) into images composed of pixels on a display device.

Conceptually, graphics acts as a translator between the digital world (numbers, algorithms) and the human visual system.

Definition

Q: Computer Graphics acts as a "translator" between which two domains?

A: The Digital World (numbers, algorithms) and the Human Visual System.

1.2 Interactive Computer Graphics

Interactive Computer Graphics refers to systems where the user can modify the displayed image in real time and immediately observe the result. Interaction closes the loop between user input and visual output.

  • Input: mouse, keyboard, touch, sensors
  • Processing: recalculation of geometry, lighting, or view
  • Output: updated image within milliseconds

Example: rotating a 3D model by dragging the mouse. Each movement triggers a new computation and a new image.

Interaction

Q: What distinguishes "Interactive" Computer Graphics from static graphics?

A: The closed feedback loop: User Input \(\rightarrow\) Instant Processing \(\rightarrow\) Updated Visual Output.

1.3 Importance of Computer Graphics

Graphics is important because vision is the fastest and richest human sensory channel. Visual representation reduces cognitive load and increases understanding.

1.3.1 Intuitive Interaction

Graphics allows users to interact naturally with systems using visual metaphors such as windows, icons, buttons, and sliders instead of abstract commands.

Intuition

Q: How does graphics reduce cognitive load?

A: By using visual metaphors (icons, windows) instead of abstract text commands.

1.3.2 Enhanced Communication

Complex ideas can be communicated quickly using graphs, diagrams, animations, and visual simulations, making abstract data concrete.

1.3.3 Scientific Visualization

Large numerical datasets become understandable when mapped to color, shape, and motion, revealing patterns that are invisible in raw numbers.

Data

Q: Why do we use Scientific Visualization for large datasets?

A: To reveal patterns and trends that are invisible in raw numerical tables.

1.3.4 Realism in Games and Movies

Graphics enables believable virtual worlds by simulating geometry, lighting, materials, and motion to match real-world perception.

1.3.5 Simulations and Training

Graphics-based simulators allow safe, repeatable training for high-risk tasks such as flying aircraft, performing surgery, or operating machinery.

Safety

Q: What is the main benefit of using graphics-based simulators for training?

A: Safety and Repeatability in high-risk environments (e.g., flight, surgery).

1.4 Applications of Computer Graphics

Computer Graphics is applied wherever visual representation improves understanding, design, or experience.

1.4.1 Entertainment

Graphics creates immersive visual experiences.

  • Animation: frame-by-frame illusion of motion
  • Visual Effects (VFX): blending real and virtual scenes
  • 3D Modeling: digital creation of objects and characters
1.4.2 Education

Graphics supports learning by visual explanation.

  • Visual Aids: diagrams, charts, animations
  • Simulators: interactive learning environments
1.4.3 Medical Imaging

Graphics converts raw scan data into interpretable images.

  • MRI: soft tissue visualization
  • CT: internal structural imaging
Medical

Q: How does computer graphics assist in Medicine?

A: By converting **invisible** raw scan data (MRI, CT) into visible, interpretable 3D models.

1.4.4 Scientific Visualization

Graphics helps scientists explore and analyze complex phenomena.

  • Climate Models: atmospheric and ocean simulations
  • Data Representation: multidimensional datasets
1.4.5 CAD/CAM

Graphics enables precise digital design and manufacturing.

  • Engineering Design: mechanical and electrical components
  • Architecture: buildings and structural layouts
CAD

Q: What is the primary role of CAD/CAM?

A: Enabling precise digital design and manufacturing of components.

1.4.6 Virtual Reality and Gaming

Graphics creates immersive, real-time interactive environments that respond to user movement and actions.

1.5 Graphics Pipeline

The Graphics Pipeline is an ordered sequence of transformations that converts a scene description into a final image. Each stage solves one specific problem in turning 3D data into 2D pixels.

The Pipeline

Q: In one sentence, what does the Graphics Pipeline do?

A: It converts a 3D scene description into a 2D final image through a sequence of steps.

1.5.1 Modeling

Defines the shape of objects using geometric primitives such as points, lines, and polygons in object space.

Pipeline Step 1

Q: In the Modeling stage, where do the objects exist?

A: In Object Space (defined by geometric primitives).

1.5.2 Transformation

Positions objects in the scene by applying translation, rotation, and scaling using matrices.

Pipeline Step 2

Q: Which mathematical tool is used for Transformation?

A: Matrices (for translation, rotation, scaling).

1.5.3 Lighting

Computes color and brightness based on light sources, surface properties, and viewing direction.

1.5.4 Projection

Converts 3D coordinates into 2D screen coordinates using parallel or perspective projection.

Pipeline Step 4

Q: What is the key dimensional change in Projection?

A: Converting 3D coordinates into 2D screen coordinates.

1.5.5 Clipping

Removes parts of objects that lie outside the viewing region to improve efficiency.

Pipeline Step 5

Q: Why is Clipping necessary?

A: To improve efficiency by discarding objects outside the view.

1.5.6 Rasterization

Converts geometric primitives into discrete pixels and assigns color values to each pixel.

Pipeline Step 6

Q: What is the output of the Rasterization stage?

A: Discrete pixels with assigned colors.

1.5.7 Display

Final pixel values are stored in the frame buffer and sent to the display device to form the visible image.

Introductory Highlights

  • Computer Graphics: Translating abstract data/numbers into visual representations (images) for human interpretation.
  • Interactive Graphics system: Closed-loop system where user input (mouse/keyboard) immediately updates the visual output in real-time.
  • Graphics Pipeline: Ordered sequence of steps to convert 3D models to 2D pixels: Modeling \(\rightarrow\) Transformation \(\rightarrow\) Lighting \(\rightarrow\) Projection \(\rightarrow\) Clipping \(\rightarrow\) Rasterization \(\rightarrow\) Display.
  • Rasterization: The specific stage of converting geometric shapes (lines/polygons) into a grid of pixels for the display.
  • Scientific Visualization: Mapping massive numerical datasets (e.g., climate data) to visual properties to reveal patterns.
  • Medical Imaging: Converting raw scan data (MRI/CT) into interpretable visual slices or models for diagnosis.
