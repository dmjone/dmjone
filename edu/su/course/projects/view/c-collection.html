Project Description

Designed and developed a comprehensive C programming framework to automate and manage lab exercises for the course Logic Building with Computer Programming (CSU1128). The project involved creating a structured and modular codebase to support over 80 different programming tasks, categorized into labs, practical files, and class exercises.

Relevant Responsibilities, Contributions, and Achievements

  • Modular Code Structure: Implemented a modular approach to organize and execute over 80 programming tasks across different categories (lab sessions, practical files, and class exercises).
  • Custom Function Development: Developed a set of custom C functions for input handling, file operations, and system-level tasks, which were used across multiple programs to streamline processes and reduce redundancy.
  • Menu-Driven Interface: Created a user-friendly, menu-driven interface that allowed students to select and execute specific lab tasks, making it easier to navigate and use the program suite.
  • Logging System: Implemented a logging mechanism to record user actions and program outputs, which facilitated tracking of activities and troubleshooting.
  • Cross-Platform Functionality: Ensured the code was compatible with both Windows and Linux systems by using appropriate system commands for clearing screens, handling inputs, and other system-specific tasks.

Technologies Used: C, GCC, Standard C Libraries (stdio.h, stdlib.h, math.h, string.h), Custom C Functions, CS50 Library

Outcome

Successfully automated the execution of lab exercises, which resulted in more efficient lab sessions and reduced manual errors. The structured and modular code allowed for easier maintenance and scalability.
