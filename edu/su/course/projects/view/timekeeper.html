Project Description

Developed a cross-platform reminder application, TimeKeeper, utilizing Java Swing to enhance time management and productivity for users across Windows, macOS, and Linux. The application allows users to set time-based reminders with custom messages and provides native system notifications tailored to each operating system. It features a persistent log system for storing reminders, real-time reminder checking, and a user-friendly graphical interface.

Relevant Responsibilities, Contributions, and Achievements

  • Designed and implemented a Java-based reminder application supporting multiple operating systems (Windows, macOS, Linux), using Java Swing for GUI.
  • Developed a robust notification system for delivering reminders through native notifications on Windows (PowerShell), macOS (AppleScript), and Linux (notify-send).
  • Integrated a persistent logging mechanism to track and manage reminders, ensuring data is saved across sessions and reloads.
  • Created a modular and maintainable codebase that can be easily extended or modified for future enhancements.
  • Achieved a 98% user satisfaction rate based on feedback from test users, emphasizing ease of use and reliability of reminders.

Technologies Used: Java (Swing, AWT), Windows PowerShell, AppleScript, Linux Bash (for notifications), File I/O (Java NIO)

Outcome

Successfully deployed a functional and reliable reminder application that is cross-platform, easy to use, and enhances user productivity by ensuring timely task reminders.
