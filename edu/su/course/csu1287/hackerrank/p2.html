Problem Statement

Active Traders

An institutional broker wants to review their book of customers to see which are most active. Given a list of trades by customer name, determine which customers account for at least 5% of the total number of traders. Ordert he list alphabatically ascending by name.

Example

n = 23
customers = ["Bigcorp", "Bigcorp", "Acme", "Bigcorp", "Zork", "Zork", "Abc", "Bigcorp", "Acme", "Bigcorp", "Bigcorp", "Zork", "Bigcorp", "Zork", "Zork", "Bigcorp", "Acme", "Bigcorp", "Acme", "Bigcorp", "Acme", "Littlecorp", "Nadircorp"].

Bigcorp had 10 trades out of 23, which is 43.48% of the total trades.

Both Acme and Zork had 5 trades, which is 21.74% of the total trades.

The Littlecorp, Nadir, and Abc had 1 trade each, which is 4.35% of the total trades.

So, the answer is ["Acme", "Bigcorp", "Zork"] (in alphabetical order) because only these three companies places at least 5% of the trades.

Concepts and Explanation

Here's a detailed explanation of each step in the second code implementation:

  1. Create an unordered_map customer_counts to store the count of trades made by each customer. This data structure allows for efficient access and manipulation of trade counts associated with customer names.

  2. Create a vector most_active_customers to store the most active customers. This vector will be used to store the names of customers who meet the 5% threshold and will be returned as the final result.

  3. Iterate through the customers list and update the count of trades in customer_counts. Using a range-based for loop, we iterate through each customer name in the input list, incrementing their associated trade count in the customer_counts unordered_map.

  4. Calculate the total number of trades by obtaining the size of the customers list. This value is stored in the total_trades variable and will be used to calculate the percentage of trades made by each customer.

  5. Iterate through customer_counts to find the customers who made at least 5% of the total trades. Using a range-based for loop, we iterate through each key-value pair (customer name and trade count) in the customer_counts unordered_map.

    • Calculate the percentage of trades made by the current customer by dividing their trade count by the total number of trades and multiplying the result by 100. This value is stored in the percentage variable.

    • If the percentage is equal to or greater than 5%, add the customer to the most_active_customers list. This step filters out customers who do not meet the 5% threshold.

  6. Sort the most_active_customers list in alphabetical order using the sort function from the C++ Standard Library. This step ensures the final list of most active customers is returned in alphabetical order.

  7. Return the sorted most_active_customers list as the final result of the function.

vector mostActive(vector customers) {
    // Create an unordered_map to store the count of trades made by each customer
    unordered_map customer_counts;
    
    // Create a vector to store the most active customers
    vector most_active_customers;
    
    // Iterate through the customers list and update the count of trades in customer_counts
    for (const string& customer : customers) {
        customer_counts[customer]++;
    }

    // Calculate the total number of trades
    int total_trades = customers.size();
    
    // Iterate through customer_counts to find the customers who made at least 5% of the total trades
    for (const auto& entry : customer_counts) {
        // Calculate the percentage of trades made by the current customer
        double percentage = (entry.second * 100.0) / total_trades;
        
        // If the percentage is equal to or greater than 5%, add the customer to the most_active_customers list
        if (percentage >= 5.0) {
            most_active_customers.push_back(entry.first);
        }
    }

    // Sort the most_active_customers list in alphabetical order
    sort(most_active_customers.begin(), most_active_customers.end());
    
    // Return the sorted most_active_customers list
    return most_active_customers;
}

string ltrim(const string &str) {
    string s(str);

    s.erase(
        s.begin(),
        find_if(s.begin(), s.end(), not1(ptr_fun(isspace)))
    );

    return s;
}

string rtrim(const string &str) {
    string s(str);

    s.erase(
        find_if(s.rbegin(), s.rend(), not1(ptr_fun(isspace))).base(),
        s.end()
    );

    return s;
}
