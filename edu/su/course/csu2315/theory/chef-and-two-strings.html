1. Problem Statement & Logic

Problem: Given two strings $S1$ and $S2$ containing lowercase letters and '?'. '?' can be any character. Find the minimum and maximum possible difference between the strings. Difference is the number of indices $i$ where $S1[i] \neq S2[i]$.

Logic:

  • Minimum Difference: We want to match characters as much as possible.
    • If $S1[i] == S2[i]$: Diff doesn't increase.
    • If either is '?': We can choose '?' to match the other. Diff doesn't increase.
    • If both are '?': We make them same. Diff doesn't increase.
    • Only if $S1[i] \neq S2[i]$ AND neither is '?': Diff MUST increase by 1.
  • Maximum Difference: We want to mismatch as much as possible.
    • If either is '?': We can choose it to be different. Diff increases.
    • If $S1[i] \neq S2[i]$: Diff increases.
    • Only if $S1[i] == S2[i]$ (and neither '?'): Diff cannot increase.

1.1 Dry Run

Example: $S1 = "a?c"$, $S2 = "??b"$

  • Index 0: 'a', '?'. Min: Match ($0$). Max: Mismatch ($1$).
  • Index 1: '?', '?'. Min: Match ($0$). Max: Mismatch ($1$).
  • Index 2: 'c', 'b'. Min: Mismatch ($1$). Max: Mismatch ($1$).
  • Result: Min $0+0+1 = 1$. Max $1+1+1=3$. Output: 1 3.

1.2 Pseudocode Solution


READ T
FOR i FROM 1 TO T:
    READ S1
    READ S2
    
    min_diff = 0
    max_diff = 0
    
    FOR j FROM 0 TO Length(S1)-1:
        c1 = S1[j]
        c2 = S2[j]
        
        # Calculate Max Diff
        IF c1 == '?' OR c2 == '?' OR c1 != c2:
            max_diff = max_diff + 1
            
        # Calculate Min Diff
        IF c1 != '?' AND c2 != '?' AND c1 != c2:
            min_diff = min_diff + 1
            
    PRINT min_diff + " " + max_diff
  • Time Complexity: $O(N)$ where $N$ is string length.
  • Space Complexity: $O(1)$.
To minimize the difference, how do we treat the '?' characters?

A: We treat them as wildcards that can match any character, so they contribute 0 to the difference.
