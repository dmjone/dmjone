1. Problem Statement & Logic

Problem: Given $N \times N$ matrix $A$. Find max trace sum of any square submatrix. (Trace = sum of diagonal elements).

Logic:

  • Wait, simple trace of *any* square submatrix.
  • Any square submatrix's diagonal is part of a diagonal of the original matrix.
  • Consider all diagonals of $A$. (Main diagonal and those parallel to it).
  • For each diagonal, finding the submatrix with max trace is equivalent to finding the "Maximum Sum Subarray" (Kadane's Algorithm) on that diagonal sequence? No.
  • The elements of a submatrix diagonal are contiguous elements of an original matrix diagonal.
  • Yes, simply run Kadane's algorithm on every diagonal of the matrix.
  • Diagonals start from first row or first column.
  • Result is $\max(\text{Kadane}(\text{diagonal}_k))$.

1.1 Dry Run

Example: 2x2: [[1, -5], [-2, 3]]

  • Diag 1 (Start 0,0): 1, 3. Contiguous subsegments: [1], [3], [1,3](4). Max 4.
  • Diag 2 (Start 0,1): -5. Max -5.
  • Diag 3 (Start 1,0): -2. Max -2.
  • Overall Max: 4.

1.2 Pseudocode Solution


READ T
FOR i FROM 1 TO T:
    READ N
    READ Matrix[N][N]
    
    max_trace = -Infinity
    
    # Check diagonals starting from row 0
    FOR c FROM 0 TO N-1:
        current_diag = []
        r = 0, cc = c
        WHILE r < N AND cc < N:
            current_diag.ADD(Matrix[r][cc])
            r++, cc++
        max_trace = MAX(max_trace, KADANE(current_diag))
        
    # Check diagonals starting from col 0 (skipping 0,0)
    FOR r FROM 1 TO N-1:
        current_diag = []
        rr = r, c = 0
        WHILE rr < N AND c < N:
            current_diag.ADD(Matrix[rr][c])
            rr++, c++
        max_trace = MAX(max_trace, KADANE(current_diag))
        
    PRINT max_trace
  • Time Complexity: $O(N^2)$.
  • Space Complexity: $O(N)$ (for diagonal storage).
Why can we apply Kadane's algorithm to matrix diagonals?

A: The trace of any square submatrix corresponds to a contiguous segment of one of the original matrix's diagonals. Finding the maximum trace is therefore equivalent to finding the Maximum Sum Subarray (Kadane's) on each diagonal sequence.
