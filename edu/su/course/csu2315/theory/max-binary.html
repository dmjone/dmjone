1. Problem Statement & Logic

Problem: Given binary string $S$, len $N$. $K$ operations. Types: 1. Add '0' anywhere. 2. Flip '0' to '1'. Maximize decimal value.

Logic:

  • To maximize value:
    • The most significant bit (index 0) should be '1'. If $S[0] == '0'$, flip it to '1'. This costs 1 flip op.
    • Remaining operations should be used to increase value (add length).
    • Adding a '0' at the END multiplies the value by 2. This is very powerful.
    • Flipping other '0's to '1's adds minimal value ($2^k$) compared to appending '0's (doubling).
    • Strategy:
      1. Ensure $S[0]$ is '1'. If not, flip it ($K = K-1$).
      2. Use all remaining $K$ ops to append '0's to the end.

1.1 Dry Run

Example: $S=$ "011", $K=2$

  • Start: "011" (3).
  • Op 1: Flip $S[0]$. $\to$ "111" (7). $K=1$.
  • Op 2: Append '0'. $\to$ "1110" (14). $K=0$.
  • Output: "1110".

1.2 Pseudocode Solution

READ T
FOR i FROM 1 TO T:
    READ N, K
    READ S
    
    IF S[0] == '0':
        S[0] = '1'
        K = K - 1
        
    PRINT S + REPEAT('0', K)
  • Time Complexity: $O(N+K)$.
  • Space Complexity: $O(N+K)$.
