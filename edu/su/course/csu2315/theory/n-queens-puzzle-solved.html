1. Problem Statement & Logic

Problem: Calculate $(0.143 \times N)^N$ rounded to the nearest integer.

Logic:

  • Use the built-in power function `pow`.
  • Base = $0.143 \times N$.
  • Exponent = $N$.
  • Result = $\text{round}( \text{pow}(\text{Base}, \text{Exponent}) )$.

1.1 Dry Run

Example: $N=4$

  • Base = $0.143 \times 4 = 0.572$.
  • Value = $0.572^4 \approx 0.107$.
  • Round(0.107) = 0. Output: 0.

Example: $N=10$

  • Base = $1.43$. $1.43^{10} \approx 36.xxx$. Output: 36 (approx).

1.2 Pseudocode Solution


READ T
FOR i FROM 1 TO T:
    READ N
    
    base = 0.143 * N
    result = POW(base, N)
    ans = ROUND(result)
    
    PRINT ans
  • Time Complexity: $O(1)$ (using pow).
  • Space Complexity: $O(1)$.
Why do we use the `pow` function with a float base?

A: The base $0.143 \times N$ is a floating-point number. Standard integer exponentiation would truncate or lose precision. We must calculate the power in floating-point arithmetic and `round` the final result.
