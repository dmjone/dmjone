1. Problem Statement & Logic

Problem: Buy $N$ burgers in budget $R$. Standard cost $X$, Premium cost $Y$. Maximize Premium burgers.

Logic:

  • Let count of premium burgers be $k$. Count of normal is $N-k$.
  • Constraint: $k \cdot Y + (N-k) \cdot X \le R$.
  • $k \cdot Y + N \cdot X - k \cdot X \le R$.
  • $k(Y - X) \le R - N \cdot X$.
  • $k \le \frac{R - N \cdot X}{Y - X}$.
  • Also $0 \le k \le N$.
  • Check if basic requirement met: $N \cdot X \le R$. If not, output -1.
  • Else, calculate max $k$ using formula.

1.1 Dry Run

Example: $X=2, Y=5, N=4, R=15$

  • Min Cost: $4 \times 2 = 8 \le 15$. Possible.
  • Max Premium: $k \le (15 - 8) / (5 - 2) = 7 / 3 = 2.33$.
  • Max integer $k=2$.
  • Normal: $4-2=2$. Premium: 2.
  • Check: $2 \times 2 + 2 \times 5 = 4 + 10 = 14 \le 15$. Valid.
  • Output: 2 2.

1.2 Pseudocode Solution

READ T
FOR i FROM 1 TO T:
    READ X, Y, N, R
    
    IF R < N * X:
        PRINT -1
    ELSE:
        premium_count = (R - (N * X)) / (Y - X)
        IF premium_count > N:
            premium_count = N
        
        # Use simple integer division, standard implementation handles logic
        # But ensure we don't exceed N
        # Actually simplest: min(N, (R - N*X)/(Y-X)) provided R>=NX
        
        normal_count = N - premium_count
        PRINT normal_count + " " + premium_count
  • Time Complexity: $O(1)$.
  • Space Complexity: $O(1)$.
