1. Problem Statement & Logic

Problem: Break a stick of length $L$ into $K$ parts. Minimize the sum of absolute differences between adjacent parts' lengths.

Logic:

  • We can choose the lengths of the parts arbitrarily as long as they sum to $L$ and are positive.
  • To minimize $|A_{i+1} - A_i|$, we should make all parts as equal as possible.
  • Specifically, if we simply make all parts equal to $L/K$ (or as close as possible), the differences will be small.

  • Okay, if $L \% K == 0$, calculate sum of diffs for equal parts (which is 0).
  • If $L \% K \neq 0$, we have some parts of size `floor(L/K)` and some `floor(L/K) + 1`.
  • We can arrange them like `[x, x, ..., x, x+1, ..., x+1]`. The only difference occurs at the boundary between $x$ and $x+1$.
  • The difference is $|(x+1) - x| = 1$. And there is only 1 such boundary if we group them.
  • So if $L \% K == 0$, answer 0. If $L \% K \neq 0$, answer 1.

1.1 Dry Run

Example: $L=4, K=3$

  • $4 \% 3 \neq 0$. Output: 1.
  • Explanation: Parts [1, 1, 2]. Diff $|1-1| + |2-1| = 0+1=1$.

Example: $L=6, K=2$

  • $6 \% 2 == 0$. Output: 0.
  • Explanation: Parts [3, 3]. Diff $|3-3|=0$.

1.2 Pseudocode Solution


READ T
FOR i FROM 1 TO T:
    READ L, K
    
    IF L % K == 0:
        PRINT 0
    ELSE:
        PRINT 1
  • Time Complexity: $O(1)$.
  • Space Complexity: $O(1)$.
Why is the answer 0 if $L$ is divisible by $K$, and 1 otherwise?

A: If divisible, we make $K$ equal parts (diff 0). If not, we use parts of size $\lfloor L/K \rfloor$ and $\lfloor L/K \rfloor + 1$. By arranging all smaller parts together and larger parts together, we create only one boundary where adjacent lengths differ by 1.
