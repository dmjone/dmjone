1. Problem Statement & Logic

Problem: Alice and Bob want to split animals such that they get identical multisets (same count of each animal type). Given $N$ animals with types $A_i$. Is it possible?

Logic:

  • If they have identical multisets, then for every animal type $t$, Alice gets $k$ and Bob gets $k$.
  • Total count of type $t$ is $2k$, which is always even.
  • So, the condition is possible if and only if the frequency of EVERY animal type is EVEN.
  • Count frequencies. If any is odd, output NO. Else YES.

1.1 Dry Run

Example: $N=4, A=[4, 4, 3, 3]$

  • Freq 4: 2 (even). Freq 3: 2 (even). Output: YES.

Example: $N=3, A=[1, 2, 3]$

  • Freq 1: 1 (odd). Output: NO.

1.2 Pseudocode Solution


READ T
FOR i FROM 1 TO T:
    READ N
    READ A (array)
    
    Freq = Map or Array
    FOR x in A:
        Freq[x] = Freq[x] + 1
        
    possible = TRUE
    FOR count in Freq.values():
        IF count % 2 != 0:
            possible = FALSE
            BREAK
            
    IF possible:
        PRINT "YES"
    ELSE:
        PRINT "NO"
  • Time Complexity: $O(N)$.
  • Space Complexity: $O(N)$.
Why must every animal frequency be even?

A: For Alice and Bob to have *identical* collections, every animal type must be split exactly equally (e.g., $k$ for Alice, $k$ for Bob). Thus, the total count for any type must be $2k$, which is always an even number.
