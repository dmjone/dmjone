1. Problem Statement & Logic

Problem: Chef plays guitar. He plucks a sequence of strings $S_1, S_2, \dots, S_N$. To move from string $S_i$ to $S_{i+1}$, he skips the strings strictly between them. Number of skipped strings = $|S_i - S_{i+1}| - 1$. Calculate total skipped strings.

Logic:

  • Initialize total skipped = 0.
  • Iterate from $i=0$ to $N-2$.
  • Add $|S_i - S_{i+1}| - 1$ to total.
  • Note: If adjacent strings are same or adjacent integers (e.g. 1 and 2), skipped is 0. The formula handles this ($|2-1|-1 = 0$). If strings are same ($|1-1|-1 = -1$), this shouldn't happen based on physical movement constraints usually, but constraints say $S_i$ can be equal? Usually formula is $\max(0, |a-b|-1)$. Logic check: "skips the strings in between". If I stay on same string, I skip 0. Formula gives -1. If problem guarantees $S_i \ne S_{i+1}$, simple formula works. If not, use $\max(0, \dots)$. Wait, usually distinct steps. Let's stick to $|x-y|-1$.

1.1 Dry Run

Example: $S = [1, 6, 11]$

  • Move 1 to 6: Skips 2, 3, 4, 5. Count = $|1-6|-1 = 5-1=4$.
  • Move 6 to 11: Skips 7, 8, 9, 10. Count = $|6-11|-1 = 5-1=4$.
  • Total: $4+4=8$.

1.2 Pseudocode Solution


READ T
FOR i FROM 1 TO T:
    READ N
    READ Array S
    
    total_skipped = 0
    
    FOR j FROM 0 TO N-2:
        diff = ABS(S[j] - S[j+1])
        skipped = diff - 1
        total_skipped = total_skipped + skipped
        
    PRINT total_skipped
  • Time Complexity: $O(N)$.
  • Space Complexity: $O(1)$.
What is the formula for calculating skipped strings between two guitar strings?

A: The absolute difference minus one: $|S_i - S_{i+1}| - 1$. For example, moving from string 1 to 6 skips 4 strings (2, 3, 4, 5).
