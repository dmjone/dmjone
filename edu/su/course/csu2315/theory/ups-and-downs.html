1. Problem Statement & Logic

Problem: Rearrange array $A$ such that $A_0 \le A_1 \ge A_2 \le A_3 \dots$. (Ups and Downs).

Logic:

  • Simple strategy: Sort the array $A$.
  • Then swap pairs $(A_1, A_2), (A_3, A_4) \dots$ (indices 1-based) OR $(A_1, A_2)$?
  • Let's trace: Sorted $S_0, S_1, S_2, S_3$.
    • $S_0 \le S_1$. Condition 1 ok.
    • $S_1$ might be $\le S_2$. Condition 2 requires $A_1 \ge A_2$.
    • Swap $S_1, S_2 \to S_0, S_2, S_1, S_3$.
    • $S_0 \le S_2$ (Yes). $S_2 \ge S_1$ (Yes). $S_1 \le S_3$ (Yes).
  • General Algorithm:
    1. Sort $A$.
    2. Iterate $i$ from 1 to $N-2$ with step 2. Swap $A[i]$ and $A[i+1]$.

1.1 Dry Run

Example: $A=[3, 1, 2, 4]$

  • Sort: $[1, 2, 3, 4]$.
  • Swap index 1,2 (values 2,3): $[1, 3, 2, 4]$.
  • Check: $1 \le 3$ (ok), $3 \ge 2$ (ok), $2 \le 4$ (ok).
  • Output: 1 3 2 4.

1.2 Pseudocode Solution

READ T
FOR i FROM 1 TO T:
    READ N
    READ A (array)
    SORT(A)
    
    FOR j FROM 1 TO N-2 STEP 2:
        SWAP(A[j], A[j+1])
        
    PRINT A
  • Time Complexity: $O(N \log N)$.
  • Space Complexity: $O(1)$.
