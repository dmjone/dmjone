1. Problem Statement & Logic

Problem: Limak and Bob compete in an eating contest. Limak can eat at most $A$ candies, and Bob can eat at most $B$ candies. They take turns eating candies in increasing order:

  • Turn 1: Limak eats 1 candy.
  • Turn 2: Bob eats 2 candies.
  • Turn 3: Limak eats 3 candies.
  • ...
  • Turn $k$: The person whose turn it is eats $k$ candies.

The game ends when a player cannot eat the required number of candies because it would exceed their limit. That player loses, and the other wins. Determine the winner.

Logic: We simulate the game round by round.

  • Keep track of `sum_limak` and `sum_bob`.
  • Iterate $i$ from 1, 2, 3, ...
  • If $i$ is odd (Limak's turn): Check if `sum_limak + i <= A`. If not, Bob wins.
  • If $i$ is even (Bob's turn): Check if `sum_bob + i <= B`. If not, Limak wins.

1.1 Dry Run

Example: $A=3, B=2$

  • Turn 1 (Limak): Needs 1. Total Limak = 1. ($1 \le 3$, OK)
  • Turn 2 (Bob): Needs 2. Total Bob = 2. ($2 \le 2$, OK)
  • Turn 3 (Limak): Needs 3. Total Limak = $1+3=4$. ($4 > 3$, FAIL)
  • Result: Bob wins.

1.2 Pseudocode Solution


READ T
FOR i FROM 1 TO T:
    READ A, B
    
    limak_eaten = 0
    bob_eaten = 0
    turn = 1
    
    WHILE TRUE:
        # Odd turn = Limak
        IF turn % 2 != 0:
            IF limak_eaten + turn <= A:
                limak_eaten = limak_eaten + turn
            ELSE:
                PRINT "Bob"
                BREAK
        
        # Even turn = Bob
        ELSE:
            IF bob_eaten + turn <= B:
                bob_eaten = bob_eaten + turn
            ELSE:
                PRINT "Limak"
                BREAK
                
        turn = turn + 1
  • Time Complexity: $O(\sqrt{A} + \sqrt{B})$. The sum of first $k$ integers grows quadratically ($k^2 \approx A$), so loops run $\approx \sqrt{A}$.
  • Space Complexity: $O(1)$.
How many turns will the game last approximately?

A: The total candies eaten grows quadratically ($\approx k^2/2$). So the game lasts roughly $\sqrt{2 \cdot \max(A, B)}$ turns, which is very fast.
