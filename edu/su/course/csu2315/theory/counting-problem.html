1. Problem Statement & Logic

Problem: Given an array $A$. Can we split it into two non-empty subsequences $S1$ and $S2$ such that sum($S1$) $\times$ sum($S2$) is Odd? Note: $S1 \cup S2 = A$ and $S1 \cap S2 = \emptyset$.

Logic:

  • Product of two numbers is Odd if and only if BOTH numbers are Odd.
  • So we need sum($S1$) to be Odd AND sum($S2$) to be Odd.
  • Total Sum of Array $A =$ sum($S1$) + sum($S2$).
  • If both sum($S1$) and sum($S2$) are Odd, their sum (Total Sum) must be Even (Odd + Odd = Even).
  • Condition 1: Total Sum of $A$ must be Even (otherwise we can't split it into two Odds).
  • Condition 2: We need to be able to form an Odd sum subsequence. An Odd sum requires at least one Odd number in the array.
  • If there is at least one odd number, and the total sum is even, it means there are an even number of odd numbers in the array (e.g., 2, 4, 6... odd numbers).
  • If we have at least 2 odd numbers (or even number of odd numbers, but > 0), we can put one odd number in $S1$ (making sum Odd) and the rest in $S2$.
  • Wait, if Total Sum is Even, and we pick $S1$ with Odd sum, then $S2 = Total - S1$. Even - Odd = Odd. So $S2$ is automatically Odd.
  • So the only requirement is: Total Sum is Even AND there is at least one Odd number (so we can make $S1$ odd). Actually, if Total Sum is Even and we have NO odd numbers, then all numbers are Even. Sum of any subset is Even. Product = Even * Even = Even. So we need at least one Odd number.
  • However, if total sum is Even, can we always make an odd subset?
    • If array has any odd number $x$:
      • If total sum is even, then the count of odd numbers is even. Let count be $2k, k \ge 1$.
      • Put one odd number in $S1$. $S1$ sum is odd.
      • Remaining numbers (including $2k-1$ odds) go to $S2$. Odd numbers in $S2$ is odd count. Sum of odd count from odd numbers is odd. Plus evens = Odd.
      • So yes.
  • Conclusion: We need Total Sum % 2 == 0 AND Count of Odd numbers > 0.
  • Actually shorter: If Total Sum is Odd, answer is NO. If Total Sum is Even, check if there's any odd number. If yes -> YES. If no (all evens) -> NO.

1.1 Dry Run

Example: $A = [1, 1, 2, 2]$

  • Sum = 6 (Even).
  • Odd numbers present? Yes (1).
  • Condition met? Yes. Output: YES.

Example: $A = [2, 4, 6]$

  • Sum = 12 (Even).
  • Odd numbers present? No.
  • Cannot split into two Odds (sums will always be even). Output: NO.

1.2 Pseudocode Solution

READ T
FOR i FROM 1 TO T:
    READ N
    READ Array A
    
    total_sum = 0
    has_odd = FALSE
    
    FOR x in A:
        total_sum = total_sum + x
        IF x % 2 != 0:
            has_odd = TRUE
            
    IF (total_sum % 2 == 0) AND has_odd:
        PRINT "YES"
    ELSE:
        PRINT "NO"
  • Time Complexity: $O(N)$.
  • Space Complexity: $O(1)$.
