Chef and Glove

1. Problem Statement & Logic

Problem: Chef has a hand with $N$ fingers of lengths $L_1, \dots, L_N$. He has a glove with $N$ sheaths of lengths $G_1, \dots, G_N$.
- Front: Finger $i$ goes to Sheath $i$. Valid if $L_i \le G_i$ for all $i$.
- Back: Finger $i$ goes to Sheath $N-i+1$ (glove flipped). Valid if $L_i \le G_{N-i+1}$ for all $i$.
Output: "front", "back", "both", or "none".

Logic:

  • Iterate $i$ from $0$ to $N-1$ to check "Front" validity.
  • Iterate $i$ from $0$ to $N-1$ to check "Back" validity.
  • Combine results:
    • True, True $\to$ "both"
    • True, False $\to$ "front"
    • False, True $\to$ "back"
    • False, False $\to$ "none"

1.1 Dry Run

Example: $L=[1, 2, 3]$, $G=[2, 3, 4]$

  • Front: $1\le2, 2\le3, 3\le4$. All True. Front OK.
  • Back ($G$ reversed is $[4, 3, 2]$): $1\le4, 2\le3, 3\le2$ (False!). Back Fail.
  • Output: front

1.2 Pseudocode Solution


READ T
FOR i FROM 1 TO T:
    READ N
    READ Array L
    READ Array G
    
    front = TRUE
    back = TRUE
    
    # Check Front
    FOR j FROM 0 TO N-1:
        IF L[j] > G[j]:
            front = FALSE
            BREAK
            
    # Check Back
    FOR j FROM 0 TO N-1:
        IF L[j] > G[N-1-j]:
            back = FALSE
            BREAK
            
    IF front AND back:
        PRINT "both"
    ELSE IF front:
        PRINT "front"
    ELSE IF back:
        PRINT "back"
    ELSE:
        PRINT "none"
  • Time Complexity: $O(N)$.
  • Space Complexity: $O(1)$.
Can the glove fit in "both" orientations?

A: Yes, if the fingers fit the sheaths in the normal (Front) order AND in the reversed (Back) order, the output is "both".
