1. Problem Statement & Logic

Problem: Given a string $S$ of length $N$, remove the longest possible subsequence such that the remaining string is a palindrome of length $> 1$. Output the length of the removed subsequence. If not possible, output -1.

Logic:

  • We want to maximize the number of removed characters.
  • This is equivalent to minimizing the number of kept characters.
  • The remaining characters must form a palindrome of length $> 1$. The shortest possible palindrome of length $> 1$ has length 2 (e.g., "aa", "bb").
  • If we can find any character that appears at least twice in $S$, we can keep those two instances to form a palindrome of length 2. The number of removed characters would be $N - 2$.
  • If all characters in $S$ are unique, we cannot form a palindrome of length $> 1$. Output -1.

1.1 Dry Run

Example 1: $S$ = "abac" ($N=4$)

  • Frequencies: 'a': 2, 'b': 1, 'c': 1.
  • 'a' appears twice. We can keep "aa".
  • Removed length = $4 - 2 = 2$. Output: 2.

Example 2: $S$ = "abcde" ($N=5$)

  • All unique. Max palindrome length is 1 (e.g., "a").
  • Length $> 1$ not possible. Output: -1.

1.2 Pseudocode Solution

READ T
FOR i FROM 1 TO T:
    READ N
    READ String S
    
    distinct_chars = Set()
    duplicate_found = FALSE
    
    FOR char in S:
        IF char in distinct_chars:
            duplicate_found = TRUE
            BREAK
        ELSE:
            ADD char to distinct_chars
            
    IF duplicate_found:
        PRINT N - 2
    ELSE:
        PRINT -1
  • Time Complexity: $O(N)$ or $O(1)$ since alphabet size is fixed (26). We stop as soon as we find a duplicate.
  • Space Complexity: $O(1)$ (max 26 chars in set).
