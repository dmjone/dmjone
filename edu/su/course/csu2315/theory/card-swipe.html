1. Problem Statement & Logic

Problem: Track people in office. Swiping card toggles status (In/Out). Given sequence of IDs swiping, find maximum occupancy.

Logic:

  • Maintain a set of people currently inside.
  • Iterate through the array of swipes.
  • For each ID:
    • If ID is not in set (Entry): Add to set. Increment current count.
    • If ID is in set (Exit): Remove from set. Decrement current count.
  • Update `max_occupancy = max(max_occupancy, current_count)`.

1.1 Dry Run

Example: IDs: [1, 2, 2, 1]

  • 1: In. Count 1. Max 1.
  • 2: In. Count 2. Max 2.
  • 2: Out. Count 1. Max 2.
  • 1: Out. Count 0. Max 2.
  • Output: 2.

1.2 Pseudocode Solution

READ T
FOR i FROM 1 TO T:
    READ N
    READ A (array)
    
    current_people = Set()
    max_count = 0
    
    FOR id in A:
        IF id NOT IN current_people:
            ADD id TO current_people
        ELSE:
            REMOVE id FROM current_people
            
        current_count = SIZE(current_people)
        IF current_count > max_count:
            max_count = current_count
            
    PRINT max_count
  • Time Complexity: $O(N)$.
  • Space Complexity: $O(N)$ (for set).
