1. Problem Statement & Logic

Problem: $N$ coins, all initially Head ($I=1$) or Tail ($I=2$). Play $N$ rounds. In round $k$, flip all coins $1 \dots k$. Guess count of heads ($Q=1$) or tails ($Q=2$) at end.
Actually, the problem statement says "In round k, flip all coins whose number is <= k". Wait.
Usually "Coin Flip" problem on CodeChef (CONFLIP) is simpler: $G$ games. In each game, $N$ coins. Initial state $I$. We flip coins.
Actually, standard CONFLIP logic: If $N$ is even, exactly $N/2$ coins are Head and $N/2$ Tail, regardless of start. If $N$ is odd, $(N-1)/2$ are one state, $(N+1)/2$ are the other.
Specifically, if $N$ is even, count of H = count of T = $N/2$.
If $N$ is odd, the coins starting as $I$ will be $(N-1)/2$ at end, and the OTHER side will be $(N+1)/2$.
Example: $N=3, I=H$. End state: T H T. (Flip 1: H $\to$ T. Flip 2: H H $\to$ T T. Flip 3: H H H $\to$ T T T). Wait, this is not standard CONFLIP rules.
Standard CONFLIP rules: "In Round K, flip all coins $1...K$".
Let's trust the pattern:

  • If $N$ is even: Answer $N/2$.
  • If $N$ is odd:
    • If we want to count the side that matches distinct start $I$ (i.e. $I=Q$): Answer $(N-1)/2$.
    • If we want to count the side different from start $I$ (i.e. $I \neq Q$): Answer $(N+1)/2$.

1.1 Dry Run

Example: $N=5, I=1 (H), Q=1 (Count H)$

  • $N$ is odd. Starting H. Counting H.
  • Because of flips, H count will be less. Answer $(5-1)/2 = 2$.

Example: $N=5, I=1 (H), Q=2 (Count T)$

  • $N$ is odd. Starting H. Counting T.
  • Tails will be dominant. Answer $(5+1)/2 = 3$.

1.2 Pseudocode Solution


READ T
FOR i FROM 1 TO T:
    READ G
    FOR j FROM 1 TO G:
        READ I, N, Q
        
        IF N % 2 == 0:
            PRINT N / 2
        ELSE:
            IF I == Q:
                PRINT (N - 1) / 2
            ELSE:
                PRINT (N + 1) / 2
  • Time Complexity: $O(1)$.
  • Space Complexity: $O(1)$.
If the number of coins N is even, how many are Heads after the game?

A: Exactly $N/2$, regardless of the initial state.
