1. Problem Statement & Logic

Problem: Given array $A$ of size $N$ and max value $M$. Construct array $B$ ($1 \le B_i \le M$) to maximize total distance $\sum |A_i - B_i|$.

Logic:

  • We want to maximize $|A_i - B_i|$ for each $i$ independently.
  • For a fixed $A_i$ and range $[1, M]$, the value of $B_i$ that maximizes distance depends on whether $A_i$ is closer to 1 or $M$.
  • Specifically, maximum distance is either $|A_i - 1|$ or $|A_i - M|$.
  • Simply choose $B_i = 1$ or $B_i = M$, whichever yields greater difference.
  • Sum these maximum differences.

1.1 Dry Run

Example: $N=2, M=6, A=[3, 5]$

  • $i=0, A[0]=3$. Options for $B[0]$: 1 or 6.
    • $|3-1|=2$.
    • $|3-6|=3$. Max is 3 (choose 6).
  • $i=1, A[1]=5$. Options for $B[1]$: 1 or 6.
    • $|5-1|=4$.
    • $|5-6|=1$. Max is 4 (choose 1).
  • Total Distance = $3 + 4 = 7$.

1.2 Pseudocode Solution


READ T
FOR i FROM 1 TO T:
    READ N, M
    READ A (array)
    
    total_dist = 0
    FOR val in A:
        dist1 = ABS(val - 1)
        dist2 = ABS(val - M)
        total_dist = total_dist + MAX(dist1, dist2)
        
    PRINT total_dist
  • Time Complexity: $O(N)$.
  • Space Complexity: $O(1)$.
Why do we only check endpoints 1 and M for $B_i$?

A: The function $f(x) = |A_i - x|$ is maximized at the boundaries of the range $[1, M]$. Thus, the optimal $B_i$ is always one of the extremes (1 or M).
