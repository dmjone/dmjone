1. Problem Statement & Logic

Problem: Given $N$ cards with numbers. You can remove any card. Goal is to be left with cards that all have the same number. Find minimum moves (removals) required.

Logic:

  • To minimize removals, we must maximize the number of cards we Keep.
  • Since all kept cards must have the same number, we should keep the cards corresponding to the number that appears most frequently (mode) in the original set.
  • Result = Total Cards ($N$) - Frequency of most frequent element.

1.1 Dry Run

Example: $N=4$, Cards = $[1, 2, 2, 3]$

  • Frequencies: 1$\to$1, 2$\to$2, 3$\to$1.
  • Max frequency is 2 (for number 2).
  • Removals = $4 - 2 = 2$. (Remove 1 and 3).

1.2 Pseudocode Solution


READ T
FOR i FROM 1 TO T:
    READ N
    READ Array A
    
    # Count frequencies of each number
    freq_map = Map()
    max_freq = 0
    
    FOR num in A:
        IF num in freq_map:
            freq_map[num] = freq_map[num] + 1
        ELSE:
            freq_map[num] = 1
            
        IF freq_map[num] > max_freq:
            max_freq = freq_map[num]
            
    PRINT N - max_freq
  • Time Complexity: $O(N)$.
  • Space Complexity: $O(N)$ for the frequency map.
To minimize removals, what must we maximize?

A: We must maximize the number of kept cards, which implies keeping all cards of the most frequent value (the mode).
