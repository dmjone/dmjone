1. Problem Statement & Logic

Problem: Given a binary string $S$ (seats). '0' is empty, '1' is occupied. Adjacent people form a group. Count total groups.

Logic:

  • We need to count connected components of '1's.
  • A new group starts whenever we encounter a '1' that was not immediately preceded by a '1'.
  • Iterate through the string:
    • If current char is '1' AND (it's the first char OR previous char was '0'): Increment group count.

1.1 Dry Run

Example: "010110111"

  • Index 0 ('0'): Skip.
  • Index 1 ('1'): Prev '0'. New Group. Count=1.
  • Index 2 ('0'): Skip.
  • Index 3 ('1'): Prev '0'. New Group. Count=2.
  • Index 4 ('1'): Prev '1'. Same Group.
  • Index 5 ('0'): Skip.
  • Index 6 ('1'): Prev '0'. New Group. Count=3.
  • ...
  • Total 3.

1.2 Pseudocode Solution


READ T
FOR i FROM 1 TO T:
    READ S
    groups = 0
    N = LENGTH(S)
    
    FOR j FROM 0 TO N-1:
        IF S[j] == '1':
            IF j == 0 OR S[j-1] == '0':
                groups = groups + 1
                
    PRINT groups
  • Time Complexity: $O(N)$.
  • Space Complexity: $O(1)$.
When does a new group of friends begin?

A: A new group starts whenever we encounter a '1' that is either at the very beginning of the string or immediately follows a '0'.
