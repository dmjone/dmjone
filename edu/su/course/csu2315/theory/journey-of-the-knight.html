1. Problem Statement & Logic

Problem: Knight at $(X1, Y1)$. Can it reach $(X2, Y2)$ in exactly 100 moves?

Logic:

  • A knight always moves from a white square to a black square or vice versa (changes color each step).
  • In an odd number of moves, the color always changes.
  • In an even number of moves (like 100), the color of the square remains the SAME.
  • Therefore, reachable if and only if $(X1, Y1)$ and $(X2, Y2)$ have the same color.
  • Board Color Logic: A square $(x, y)$ is black if $(x+y)$ is even (or odd, consistency matters). It is white if $(x+y)$ is odd.
  • Condition: $(X1+Y1) \% 2 == (X2+Y2) \% 2$.
  • Also need to consider if distance is reachable in 100 moves? 100 is very large for 8x8 board. Knight can toggle back and forth. Only parity matters.

1.1 Dry Run

Example: Start (1,1), End (1,1).

  • $1+1=2$ (Even). End $1+1=2$ (Even). Same parity. Can reach (move away and back 50 times). Output: YES.

Example: Start (1,1), End (1,2).

  • $1+1=2$ (Even). $1+2=3$ (Odd). Different parity. Impossible in even moves. Output: NO.

1.2 Pseudocode Solution

READ T
FOR i FROM 1 TO T:
    READ X1, Y1, X2, Y2
    
    start_parity = (X1 + Y1) % 2
    end_parity = (X2 + Y2) % 2
    
    IF start_parity == end_parity:
        PRINT "YES"
    ELSE:
        PRINT "NO"
  • Time Complexity: $O(1)$.
  • Space Complexity: $O(1)$.
