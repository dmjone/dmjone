1. Problem Statement & Logic

Problem: Given array $A$. Find the length of the longest subarray with non-zero product.

Logic:

  • A product of a subarray is zero if and only if it contains at least one '0'.
  • So, the problem is equivalent to finding the longest subarray that does not contain any '0'.
  • This reduces to finding the maximum length of consecutive non-zero elements.
  • Iterate through the array:
    • If current element $\neq 0$: Increment current length counter.
    • If current element $== 0$: Reset current length counter to 0.
    • Update max length at each step.

1.1 Dry Run

Example: $A=[1, 0, 2, 3, 0, 4]$

  • 1: count=1. max=1.
  • 0: count=0.
  • 2: count=1. max=1.
  • 3: count=2. max=2.
  • 0: count=0.
  • 4: count=1. max=2.
  • Output: 2.

1.2 Pseudocode Solution


READ N
READ A (array)

max_len = 0
current_len = 0

FOR x in A:
    IF x != 0:
        current_len = current_len + 1
        IF current_len > max_len:
            max_len = current_len
    ELSE:
        current_len = 0
        
PRINT max_len
  • Time Complexity: $O(N)$.
  • Space Complexity: $O(1)$.
Why is finding a non-zero product equivalent to avoiding zeros?

A: Because a single zero in a subarray causes the entire product to be zero. Therefore, maximizing the product subarray length is exactly finding the longest contiguous segment that contains no zeros.
