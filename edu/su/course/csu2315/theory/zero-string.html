Zero String

1. Problem Statement & Logic

Problem: Given binary string $S$. Operations: 1. Delete a char. 2. Flip all chars. Min operations to make string all '0's (or empty? Problem usually implies reducing to all zeros).
Typically: "Make the string consist of ONLY 0s".

Logic:

  • We can achieve the goal in two main ways:
    1. No Flip: Just delete all '1's. Cost = Count('1').
    2. With Flip: Flip the string first (Cost 1). Then '0's become '1's and '1's become '0's. Now delete all NEW '1's (which were originally '0's). Cost = 1 + Count('0').
  • Min operations = $\min(\text{Count}(1), 1 + \text{Count}(0))$.

1.1 Dry Run

Example: "1101"

  • Count(1) = 3. Count(0) = 1.
  • Option 1 (No Flip): Delete 3 '1's. Cost 3. Result "0".
  • Option 2 (Flip): Flip $\to$ "0010" (cost 1). Delete 1 '1' (cost 1). Total 2. Result "000".
  • Min(3, 2) = 2. Output: 2.

Example: "0000"

  • Count(1) = 0. Count(0) = 4.
  • Option 1: 0. Option 2: 1 + 4 = 5.
  • Min(0, 5) = 0. Output: 0.

1.2 Pseudocode Solution


READ T
FOR i FROM 1 TO T:
    READ N
    READ S
    
    count_1 = 0
    count_0 = 0
    
    FOR char in S:
        IF char == '1': count_1 = count_1 + 1
        ELSE: count_0 = count_0 + 1
        
    ans = MIN(count_1, 1 + count_0)
    PRINT ans
  • Time Complexity: $O(N)$.
  • Space Complexity: $O(1)$.
Why is the answer $\min(\text{cnt}_1, 1 + \text{cnt}_0)$?

A: We have two strategies: 1) Delete all '1's directly. 2) Flip the string (cost 1) so '0's become '1's, then delete those new '1's (original '0's). We take the minimum cost of these two approaches.
