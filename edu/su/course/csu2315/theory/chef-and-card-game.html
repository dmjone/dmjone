1. Problem Statement & Logic

Problem: Chef and Morty play a card game of $N$ rounds. In each round both play a card. The power of a card is the sum of digits of the number on it. The player with higher power gets 1 point. If powers are equal, both get 1 point. Determine the winner (0 for Chef, 1 for Morty, 2 for Draw) and the winner's score.

Logic:

  • Helper function `power(k)`: calculates sum of digits of $k$.
  • Initialize scores `chef_score = 0`, `morty_score = 0`.
  • Loop $N$ times:
    • Read cards $A$ and $B$.
    • $P_A = power(A)$, $P_B = power(B)$.
    • If $P_A > P_B$: `chef_score++`.
    • If $P_B > P_A$: `morty_score++`.
    • If $P_A == P_B$: `chef_score++`, `morty_score++`.
  • Compare final scores to determine winner.

1.1 Dry Run

Example: $N=2$. Round 1: (10, 4). Round 2: (8, 12).

  • Round 1: Chef 10 (sum=1), Morty 4 (sum=4). $4 > 1$. Morty wins. Scores: C=0, M=1.
  • Round 2: Chef 8 (sum=8), Morty 12 (sum=3). $8 > 3$. Chef wins. Scores: C=1, M=1.
  • Final: 1 vs 1. Draw. Output: 2 1.

1.2 Pseudocode Solution


FUNCTION get_digit_sum(num):
    sum = 0
    WHILE num > 0:
        sum = sum + (num % 10)
        num = num / 10
    RETURN sum

READ T
FOR i FROM 1 TO T:
    READ N
    chef_points = 0
    morty_points = 0
    
    FOR j FROM 1 TO N:
        READ A, B
        powA = get_digit_sum(A)
        powB = get_digit_sum(B)
        
        IF powA > powB:
            chef_points = chef_points + 1
        ELSE IF powB > powA:
            morty_points = morty_points + 1
        ELSE:
            chef_points = chef_points + 1
            morty_points = morty_points + 1
            
    IF chef_points > morty_points:
        PRINT "0 " + chef_points
    ELSE IF morty_points > chef_points:
        PRINT "1 " + morty_points
    ELSE:
        PRINT "2 " + chef_points
  • Time Complexity: $O(N \times \log(\max(A, B)))$ (log factor for digit sum).
  • Space Complexity: $O(1)$.
What happens if both cards have equal power?

A: Both Chef and Morty receive 1 point each.
