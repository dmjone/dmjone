Processing a string

1. Problem Statement & Logic

Problem: Given an alphanumeric string. Find the sum of all digits present in the string.

Logic:

  • Iterate through each character of the string.
  • Check if the character is a digit ('0' to '9').
  • If it is, convert it to an integer and add to sum.
  • Ignore non-digit characters.
  • Print sum.

1.1 Dry Run

Example: "ab123x"

  • 'a': ignore.
  • 'b': ignore.
  • '1': add 1. Sum = 1.
  • '2': add 2. Sum = 3.
  • '3': add 3. Sum = 6.
  • 'x': ignore.
  • Total Sum: 6.

1.2 Pseudocode Solution

READ T
FOR i FROM 1 TO T:
    READ S
    total_sum = 0
    
    FOR char in S:
        IF char >= '0' AND char <= '9':
            digit = ASCII(char) - ASCII('0')
            total_sum = total_sum + digit
            
    PRINT total_sum
  • Time Complexity: $O(N)$.
  • Space Complexity: $O(1)$.
