1. Problem Statement & Logic

Problem: Given 3 lengths: $X, Y, Z$. Can they denote displacements in 1D such that $A \to B$ is $X$, $B \to C$ is $Y$, and $C \to A$ is $Z$?
(Actually, problem is: Can we find integers $a, b, c$ such that $|a-b|=X, |b-c|=Y, |c-a|=Z$?)

Logic:

  • On a 1D line, if we move $X$ then $Y$, the net displacement $Z$ must be either $X+Y$ (same direction) or $|X-Y|$ (opposite directions).
  • Also consider order permutations. Effectively, one length must be the sum of the other two.
  • Check if $X+Y=Z$ OR $X+Z=Y$ OR $Y+Z=X$.
  • Output "yes" if any condition holds, else "no".

1.1 Dry Run

Example: $X=1, Y=2, Z=3$

  • $1+2=3$. Condition holds ($X+Y=Z$). Output: yes.

Example: $X=1, Y=2, Z=4$

  • $1+2 \neq 4$. $1+4 \neq 2$. $2+4 \neq 1$. Output: no.

1.2 Pseudocode Solution

READ T
FOR i FROM 1 TO T:
    READ X, Y, Z
    
    IF X + Y == Z OR X + Z == Y OR Y + Z == X:
        PRINT "yes"
    ELSE:
        PRINT "no"
  • Time Complexity: $O(1)$.
  • Space Complexity: $O(1)$.
