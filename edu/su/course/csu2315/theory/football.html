1. Problem Statement & Logic

Problem: Given performance of $N$ players in a match. Arrays $A$ (goals) and $B$ (fouls). Score calculation: Multiply goals by 20, subtract fouls by 10. If score < 0, it becomes 0. Find the maximum score.

Logic:

  • Iterate through each player $i$ from $0$ to $N-1$.
  • Calculate raw score $S_i = (A[i] \times 20) - (B[i] \times 10)$.
  • Apply floor: If $S_i < 0$, set $S_i=0$.
  • Track the maximum $S_i$ found so far.

1.1 Dry Run

Example: $N=2$, $A=[2, 1]$, $B=[1, 5]$

  • Player 0: Goals 2, Fouls 1. Score $2 \times 20 - 1 \times 10 = 40 - 10 = 30$. (Max = 30).
  • Player 1: Goals 1, Fouls 5. Score $1 \times 20 - 5 \times 10 = 20 - 50 = -30$.
    • Score < 0, so Score=0.
  • Max score is 30. Output: 30.

1.2 Pseudocode Solution


READ T
FOR i FROM 1 TO T:
    READ N
    READ A (array of goals)
    READ B (array of fouls)
    
    max_score = 0
    
    FOR j FROM 0 TO N-1:
        current_score = (A[j] * 20) - (B[j] * 10)
        IF current_score < 0:
            current_score = 0
            
        IF current_score > max_score:
            max_score = current_score
            
    PRINT max_score
  • Time Complexity: $O(N)$.
  • Space Complexity: $O(1)$.
Why do we need the condition `if score < 0: score=0`?

A: The problem specifies a floor for the score. Since fouls subtract points, a player could theoretically have a negative score, but the rules dictate the minimum possible score is 0.
