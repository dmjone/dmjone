1. Problem Statement & Logic

Problem: Given four positive integers $a, b, c, d$. Determine if they can be rearranged to form a proportion. A proportion means the ratio of the first two equals the ratio of the last two (e.g., $x/y = z/w \implies x \times w = y \times z$).

Logic:

  • If the four numbers are sorted as $x \le y \le z \le w$, the only way they can form a proportion is if the smallest/middle1 equals middle2/largest.
  • Specifically, $x \times w = y \times z$.
  • Why? Because for $x/y = z/w$, we need the product of extremes to equal the product of means. To balance the product, we typically pair the smallest with the largest and the two middle numbers together.
  • Alternative: Simply check if $a \times d = b \times c$ OR $a \times c = b \times d$ OR $a \times b = c \times d$ for all permutations. But sorting simplifies it to one check: `sorted[0] * sorted[3] == sorted[1] * sorted[2]`.
  • Wait, is it strictly `sorted[0]*sorted[3]`? Consider 1, 4, 2, 2. Sorted: 1, 2, 2, 4. $1 \times 4 = 4, 2 \times 2 = 4$. Yes.
  • Consider 2, 4, 4, 8. Sorted: 2, 4, 4, 8. $2 \times 8 = 16, 4 \times 4 = 16$. Yes.

1.1 Dry Run

Example: 1, 2, 4, 2

  • Sort: 1, 2, 2, 4.
  • Check: $1 \times 4$ vs $2 \times 2$.
  • $4 == 4$. Output: Possible.

Example: 1, 2, 3, 4

  • Sort: 1, 2, 3, 4.
  • Check: $1 \times 4$ vs $2 \times 3$.
  • $4 \neq 6$. Output: Impossible.

1.2 Pseudocode Solution


READ a, b, c, d
Put elements in list and SORT them -> [s1, s2, s3, s4]

IF s1 * s4 == s2 * s3:
    PRINT "Possible"
ELSE:
    PRINT "Impossible"
  • Time Complexity: $O(1)$.
  • Space Complexity: $O(1)$.
To check if 4 numbers form a proportion, why do we sort them first?

A: Sorting simplifies the check. In a sorted sequence $a \le b \le c \le d$, we only need to verify if the product of the extremes ($a \times d$) equals the product of the means ($b \times c$).
