1.2 Pseudocode Solution

READ T
FOR i FROM 1 TO T:
    READ N
    P = Array of size N
    left = 1
    right = N
    
    # Fill from end backwards
    FOR j FROM N-1 DOWNTO 0:
        IF (N - 1 - j) % 2 == 0:
            P[j] = right
            right = right - 1
        ELSE:
            P[j] = left
            left = left + 1
            
    PRINT P
  • Time Complexity: $O(N)$.
  • Space Complexity: $O(1)$.
