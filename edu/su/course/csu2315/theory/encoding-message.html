1. Problem Statement & Logic

Problem: Encode string $S$ (length $N$) in two steps:
1. Swap valid pairs of characters: $(S_1, S_2)$, $(S_3, S_4)$, etc. If $N$ is odd, last char is not swapped.
2. Replace each character with its alphabet opposite: $a \to z, b \to y, \dots, z \to a$. Formula: NewChar = 'z' - (OldChar - 'a').

Logic:

  • Step 1 (Swap): Iterate $i$ from 0 to $N-2$ with step 2. Swap $S[i]$ and $S[i+1]$.
  • Step 2 (Replace): Iterate through all characters. Let char be $c$. New $c$ = 'a' + 'z' - $c$ ($122 - \text{code} + 97$). Simplified: 'z' - (char - 'a').
  • Do both steps sequentially or combine them.

1.1 Dry Run

Example: $S=$ "sharechat" ($N=9$)

  • Swap: (s,h), (a,r), (e,c), (h,a), (t). $\to$ "hsraceaht".
  • Replace:
    • h $\to$ s
    • s $\to$ h
    • r $\to$ i
    • a $\to$ z
    • c $\to$ x
    • e $\to$ v
    • a $\to$ z
    • h $\to$ s
    • t $\to$ g
  • Result: "shizxvzsg".

1.2 Pseudocode Solution

READ T
FOR i FROM 1 TO T:
    READ N
    READ S (as char array)
    
    # Step 1: Swap
    FOR j FROM 0 TO N-2 STEP 2:
        temp = S[j]
        S[j] = S[j+1]
        S[j+1] = temp
        
    # Step 2: Replace
    FOR k FROM 0 TO N-1:
        # 'a' + 'z' - char
        S[k] = CHAR(122 - (ASCII(S[k]) - 97))
        
    PRINT S
  • Time Complexity: $O(N)$.
  • Space Complexity: $O(N)$ (if string is immutable, else $O(1)$).
