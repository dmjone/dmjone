1. Problem Statement & Logic

Problem: RCB needs $X$ points from $Y$ remaining matches. A win gives 2 points, a tie gives 1 point, and a loss gives 0 points. We must find the minimum number of matches RCB needs to win.

Logic:

  • Let the number of wins be $W$.
  • To minimize wins, we should try to get maximum points without winning.
  • The best non-win result is a tie, which gives 1 point.
  • If RCB wins $W$ matches, then the remaining $(Y - W)$ matches can be ties.
  • Total points obtained in this best case:

    $$2W + 1(Y - W) = W + Y$$

  • To qualify, total points must be at least $X$:

    $$W + Y \ge X$$

  • This gives:

    $$W \ge X - Y$$

  • The value $(X - Y)$ represents the points that cannot be covered by ties.
  • Since each win gives 2 points, the minimum number of wins required is:

    $$W = \lceil (X - Y) / 2 \rceil$$

  • If $X \le Y$, then ties alone are sufficient and no wins are needed.

1.1 Dry Run

Example 1: $X = 10, Y = 5$

  • Maximum points using only ties = 5
  • Remaining points needed = 10 − 5 = 5
  • Each win gives 2 points
  • Wins needed = ⌈5 / 2⌉ = 3
  • But only 5 matches exist, so all matches must be wins
  • Output = 5

Example 2: $X = 1, Y = 5$

  • Maximum points using ties = 5
  • This already covers the required points
  • Output = 0

1.2 Pseudocode Solution


READ T
FOR i FROM 1 TO T:
    READ X, Y
    IF X <= Y:
        PRINT 0
    ELSE:
        remaining_points = X - Y
        wins_needed = CEIL(remaining_points / 2)
        PRINT wins_needed
  • Time Complexity: $O(1)$ per test case.
  • Space Complexity: $O(1)$.
Why is the minimum wins $X - Y$?

A: To minimize wins, we maximize points from ties. Score $2W + 1(Y-W) = W+Y$. For $W+Y \ge X$, we need $W \ge X-Y$.
