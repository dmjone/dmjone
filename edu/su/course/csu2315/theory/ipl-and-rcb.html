1. Problem Statement & Logic

Problem: RCB needs $X$ points in $Y$ matches. Win=2, Tie=1, Loss=0. Minimum wins needed?

Logic:

  • We want to minimize wins ($W$). Remaining matches can be ties ($T$) or losses ($L$).
  • To minimize $W$, we should maximize points from non-wins. The best non-win is a Tie (1 point).
  • Equation: $2W + 1(Y-W) \ge X$. (Assuming maximum points from remaining matches $Y-W$).
  • $2W + Y - W \ge X \Rightarrow W + Y \ge X \Rightarrow W \ge X - Y$.
  • So minimum wins $W = X - Y$.
  • Constraints checks:
    • If $X \le Y$, then even 0 wins (all ties) gives $Y$ points which covers $X$. So wins needed is 0. (Formula $X-Y$ gives $\le 0$).
    • Since we must answer non-negative wins: $\max(0, X - Y)$.

1.1 Dry Run

Example: $X=10, Y=5$

  • $W \ge 10 - 5 = 5$. Output: 5. (Win all).

Example: $X=1, Y=5$

  • $W \ge 1 - 5 = -4$. Output: 0.

1.2 Pseudocode Solution


READ T
FOR i FROM 1 TO T:
    READ X, Y
    wins_needed = X - Y
    IF wins_needed < 0:
        wins_needed = 0
    PRINT wins_needed
  • Time Complexity: $O(1)$.
  • Space Complexity: $O(1)$.
Why is the minimum wins $X - Y$?

A: To minimize wins, we maximize points from ties. Score $2W + 1(Y-W) = W+Y$. For $W+Y \ge X$, we need $W \ge X-Y$.
