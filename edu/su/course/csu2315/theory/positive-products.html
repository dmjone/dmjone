Positive Products

2025, Dec 15
View Problem Online

1. Problem Statement & Logic

Problem: Count pairs $(i, j)$ with $i < j$ such that $A[i] \times A[j]> 0$.

Logic:

  • Product is positive if:
    • Both numbers are positive ($+ \times + = +$).
    • Both numbers are negative ($- \times - = +$).
  • It is zero or negative if one is 0, or signs differ.
  • Count positive numbers $P$ and negative numbers $N_{eg}$.
  • Pairs with positives: $P \times (P-1) / 2$.
  • Pairs with negatives: $N_{eg} \times (N_{eg}-1) / 2$.
  • Total pairs = Sum of above.

1.1 Dry Run

Example: $A=[1, -2, 3, -4, 0]$

  • Positive: 1, 3. Count 2. Pairs $2 \times 1 / 2 = 1$.
  • Negative: -2, -4. Count 2. Pairs $2 \times 1 / 2 = 1$.
  • Zero: 0. Ignored.
  • Total: 2.

1.2 Pseudocode Solution


READ T
FOR i FROM 1 TO T:
    READ N
    READ A (array)
    
    pos_count = 0
    neg_count = 0
    
    FOR x in A:
        IF x > 0: pos_count = pos_count + 1
        IF x < 0: neg_count = neg_count + 1
        
    ans = (pos_count * (pos_count - 1)) / 2 + (neg_count * (neg_count - 1)) / 2
    PRINT ans
  • Time Complexity: $O(N)$.
  • Space Complexity: $O(1)$.
Why do we ignore zeros?

A: The condition is strictly positive ($> 0$). Any number multiplied by 0 results in 0, which is not positive. Thus, zeros cannot contribute to a valid pair.
