Even-tual Reduction

2025, Dec 15
View Problem Online

1. Problem Statement & Logic

Problem: Given a string $S$ of length $N$. You can erase any substring where every character appears an even number of times. Can you delete the entire string?

Logic:

  • If the entire string can be deleted, it means the entire string itself must satisfy the condition (or can be decomposed into parts that do).
  • Actually, the condition is necessary and sufficient: The entire string can be erased if and only if every character in the string appears an even number of times in the total string.
  • Why?
    • If every char has even count, we can just select the whole string (it is a substring of itself) and validly delete it.
    • If any char has odd count, we can never eliminate that last single instance (or odd remainder) because each deletion removes an even count. (Odd - Even = Odd). You can never reach 0.
  • So, count frequency of each char. If all are even, output "YES", else "NO".

1.1 Dry Run

Example: $S=$ "acabbad" ($N=7$)

  • Freq: a:3, b:2, c:1, d:1.
  • 'a' is odd (3). 'c' is odd (1). 'd' is odd (1).
  • Output: NO.

Example: $S=$ "aabbcc" ($N=6$)

  • Freq: a:2, b:2, c:2. All even.
  • Output: YES.

1.2 Pseudocode Solution

READ T
FOR i FROM 1 TO T:
    READ N
    READ S
    
    # Frequency array for 26 lowercase letters
    Freq = [0] * 26
    
    FOR char in S:
        index = ASCII(char) - ASCII('a')
        Freq[index] = Freq[index] + 1
        
    possible = TRUE
    FOR count in Freq:
        IF count % 2 != 0:
            possible = FALSE
            BREAK
            
    IF possible:
        PRINT "YES"
    ELSE:
        PRINT "NO"
  • Time Complexity: $O(N)$.
  • Space Complexity: $O(1)$ (fixed size alphabet).
