1. Problem Statement & Logic

Problem: Train berths pattern: 1LB, 2MB, 3UB, 4LB, 5MB, 6UB, 7SL, 8SU. Repeating every 8. Given seat $N$, find partner (same compartment/side).
Partners: (1,4), (2,5), (3,6), (7,8).

Logic:

  • Find position within block of 8: $rem = (N-1) \% 8$.
  • Map:
    • 0 (1): partner is 3 (4). Type 4LB.
    • 1 (2): partner is 4 (5). Type 5MB.
    • 2 (3): partner is 5 (6). Type 6UB.
    • 3 (4): partner is 0 (1). Type 1LB.
    • 4 (5): partner is 1 (2). Type 2MB.
    • 5 (6): partner is 2 (3). Type 3UB.
    • 6 (7): partner is 7 (8). Type 8SU.
    • 7 (8): partner is 6 (7). Type 7SL.
  • Calculate partner number mathematically or use offsets. e.g., 1+3=4. 2+3=5. 3+3=6. 7+1=8.
  • Output partner number and code (LB/MB/UB/SL/SU).

1.1 Dry Run

Example: $N=1$

  • (1 - 1) % 8 = 0. Range 0. Partner offset +3. $1+3=4$. Type 4LB. Output: 4LB.

Example: $N=5$

  • (5-1)%8 = 4. Range 4. Partner offset -3. $5-3=2$. Type 2MB. Output: 2MB.

1.2 Pseudocode Solution

READ T
FOR i FROM 1 TO T:
    READ N
    rem = (N - 1) % 8
    
    partner = 0
    type = ""
    
    SWITCH rem:
        CASE 0: partner = N + 3; type = "LB"
        CASE 1: partner = N + 3; type = "MB"
        CASE 2: partner = N + 3; type = "UB"
        CASE 3: partner = N - 3; type = "LB"
        CASE 4: partner = N - 3; type = "MB"
        CASE 5: partner = N - 3; type = "UB"
        CASE 6: partner = N + 1; type = "SU"
        CASE 7: partner = N - 1; type = "SL"
        
    PRINT partner + type
  • Time Complexity: $O(1)$.
  • Space Complexity: $O(1)$.
