1. Problem Statement & Logic

Problem: Chef wants to cover a distance `D` in steps. He can take steps of length `K`. He wants to know if he can cover the exact distance `D` using only steps of length `K`. If yes, great. If not, the remaining distance (which is less than `K`) is covered by "smaller" steps. Wait. Let's re-read the specific problem `CHEFSTEP`.

Correction: The problem `CHEFSTEP` usually refers to: "Chef has to visit $N$ friends. Distance to each friend is $D_i$. Chef's step length is $K$. Can he cover distance $D_i$ in exact steps?" i.e., is $D_i$ divisible by $K$?
Input: $N$, $K$, then array $D$.
Output: A string of 1s and 0s. 1 if divisible, 0 if not.

Logic:

  • For each distance $D_i$ in array $D$:
  • Check if $D_i \% K == 0$.
  • If yes, append '1' to result. Else '0'.

1.1 Dry Run

Example: $N=5, K=5$, $D=[10, 12, 5, 20, 11]$

  • 10 % 5 == 0? Yes (1)
  • 12 % 5 == 0? No (0)
  • 5 % 5 == 0? Yes (1)
  • 20 % 5 == 0? Yes (1)
  • 11 % 5 == 0? No (0)
  • Output: 10110

1.2 Pseudocode Solution


READ T
FOR i FROM 1 TO T:
    READ N, K
    READ Array D
    
    FOR dist in D:
        IF dist % K == 0:
            PRINT "1" (no newline)
        ELSE:
            PRINT "0" (no newline)
            
    PRINT newline
  • Time Complexity: $O(N)$ per test case.
  • Space Complexity: $O(1)$.
How do we report the answer for multiple distances?

A: By printing a continuous string of '1's (divisible) and '0's (not divisible) corresponding to each distance, without any spaces between them.
