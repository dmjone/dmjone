1. Problem Statement & Logic

Problem: Given $A$ and $B$. Add same $X \ge 0$ to both such that $A+X$ divides $B+X$. Is it possible? Output YES/NO.

Logic:

  • We need $(B+X) \% (A+X) == 0$.
  • $B+X = k(A+X)$ for some integer $k \ge 1$.
  • Difference $D = (B+X) - (A+X) = B - A$.
  • Logic: If $A$ divides $B$, $X=0$ works. YES.
  • If not, note that $A+X$ must divide the difference $D = B-A$.
  • Since $A+X > A$, we check if there exists a divisor $d$ of $B-A$ such that $d \ge A$? Or specifically, we can simply say if $X$ can be chosen.
  • Actually, we just need $B-A$ to be divisible by $A+X$.
  • This means $B-A$ (which is constant relative to $X$) must be non-zero and $(A+X)$ must be a factor of $(B-A)$.
  • Is it always possible? No. If $A=B$, diff is 0, divisible? Yes $X=0$.
  • If $A > B$, impossible (given $A \le B$).
  • If $B-A = A$, YES (already multiple).
  • If $B-A < A$, then $A+X> B-A$, so $A+X$ cannot divide $B-A$ unless $B-A=0$. So if difference is less than $A$, NO.
  • If $B-A \ge A$, we can always pick $X$ such that $A+X = B-A$ (i.e., $X = B - 2A$). Then $(A+X)$ divides $(B+X)$ because $(B+X) = 2(A+X)$. Yes.
  • Condition: YES if $B \% A == 0$ OR ($B > 2A$ or specifically $B-A \ge A$).
  • Refined condition: YES if $(B-A) \ge A$ or $A=B$. (Note if $A$ divides $B$, $B=kA$. Diff $(k-1)A \ge A$ if $k \ge 2$. If $k=1, A=B$).
  • Actually, simply: Is it possible? YES if $2A \le B$ or $A=B$ (or $A$ divides $B$, wait).
  • If $2A \le B$, we can set $X = B - 2A$. Then $A+X = B-A$. $B+X = 2B - 2A = 2(B-A)$. Divides!
  • If $A=B$, YES.
  • If $A < B < 2A$, then $0 < B-A < A$. Since $A+X \ge A> B-A$, $A+X$ cannot divide $B-A$.
  • So condition: YES if $B \ge 2A$ or $A=B$. NO otherwise. (Also check if $A$ divides $B$ initially? if $A$ divides $B$, it's YES. Does $A|B$ imply $B \ge 2A$ or $A=B$? Yes, unless $B=0$ which is not allowed usually).

1.1 Dry Run

Example: $A=3, B=6$

  • $2A = 6 \le 6$. YES. ($X=0 \to 3|6$).

Example: $A=4, B=14$

  • $2A = 8 \le 14$. YES. ($X = 14 - 8 = 6. 4+6=10, 14+6=20. 10|20$).

Example: $A=5, B=6$

  • $2A = 10 > 6$. NO.

1.2 Pseudocode Solution


READ T
FOR i FROM 1 TO T:
    READ A, B
    
    IF B % A == 0:
        PRINT "YES"
    ELSE IF B >= 2 * A:
        PRINT "YES"
    ELSE:
        PRINT "NO"
  • Time Complexity: $O(1)$.
  • Space Complexity: $O(1)$.
Why is the condition $B \ge 2A$ sufficient for YES?

A: If $B \ge 2A$, we can choose $X = B - 2A$. This makes the divisor $A+X = B-A$. The dividend becomes $B+X = 2(B-A)$. Since the dividend is exactly twice the divisor, it is divisible.
