Count of Maximum

2025, Dec 15
View Problem Online

1. Problem Statement & Logic

Problem: Given array $A$. Find the element with maximum frequency. If tie, prefer the smaller value. Output value and count.

Logic:

  • Traverse array and maintain a frequency map (count of each number).
  • Iterate through the map to find the max frequency.
  • Update `max_freq` and `best_val`.
  • Handling tie: If `current_freq > max_freq` UPDATE. If `current_freq == max_freq`, update if `current_val < best_val`.
  • Actually, traversing the map keys in sorted order or simply checking conditions works.

1.1 Dry Run

Example: $A=[1, 2, 2, 3, 3, 3]$

  • Freqs: 1:1, 2:2, 3:3.
  • Max freq is 3 (value 3). Output: 3 3.

Example: $A=[1, 1, 2, 2]$

  • Freqs: 1:2, 2:2.
  • Tie. Smaller value is 1. Output: 1 2.

1.2 Pseudocode Solution

READ T
FOR i FROM 1 TO T:
    READ N
    READ A (array)
    
    Freq = Map (default 0)
    FOR x in A:
        Freq[x] = Freq[x] + 1
        
    best_val = Infinity
    max_freq = -1
    
    FOR val, count in Freq:
        IF count > max_freq:
            max_freq = count
            best_val = val
        ELSE IF count == max_freq:
            IF val < best_val:
                best_val = val
                
    PRINT best_val + " " + max_freq
  • Time Complexity: $O(N)$.
  • Space Complexity: $O(N)$.
