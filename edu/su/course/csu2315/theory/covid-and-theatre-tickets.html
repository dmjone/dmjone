1. Problem Statement & Logic

Problem: Given a theatre with $N$ rows and $M$ seats per row. To maximize profit:
1. In a row, no two people sit adjacent. (Must handle: Seat 1 occupied $\implies$ Seat 2 Empty).
2. No two occupied rows can be adjacent. (Row 1 occupied $\implies$ Row 2 Empty).
Calculate max tickets.

Logic:

  • Row Constraints: If a row has $M$ seats, we can fill at most $\lceil M/2 \rceil$ seats.Pattern: 1, 0, 1, 0...
    Example $M=3$: [1, 0, 1] $\to$ 2 seats. Formula: $(M+1)/2$ (integer division).
  • Aisle Constraints: We can only use alternate rows. If we have $N$ rows, we can use $\lceil N/2 \rceil$ rows. Formula: $(N+1)/2$.
  • Total Profit: Seats per usable row $\times$ Usable rows.
  • Result = $((M+1)/2) \times ((N+1)/2)$.

1.1 Dry Run

Example: $N=3, M=3$

  • Usable rows: (3+1)/2 = 2. (Rows 1 and 3).
  • Seats per row: (3+1)/2 = 2. (Seats 1 and 3).
  • Total = $2 \times 2 = 4$.

Example: $N=4, M=4$

  • Usable rows: (4+1)/2 = 2. (Rows 1 and 3).
  • Seats per row: (4+1)/2 = 2. (Seats 1 and 3).
  • Total = $2 \times 2 = 4$.

1.2 Pseudocode Solution


READ T
FOR i FROM 1 TO T:
    READ N, M
    
    rows = (N + 1) / 2
    seats_per_row = (M + 1) / 2
    
    total = rows * seats_per_row
    PRINT total
  • Time Complexity: $O(1)$.
  • Space Complexity: $O(1)$.
How do we calculate the max usable seats in a row of size M?

A: Using the formula $(M + 1) // 2$, which effectively counts seats at indices 1, 3, 5... skipping adjacent ones.
