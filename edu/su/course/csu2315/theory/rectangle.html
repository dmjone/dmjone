1. Problem Statement & Logic

Problem: Given four integers $a, b, c, d$. Determine if they can form a rectangle.

Logic:

  • A rectangle has opposite sides equal.
  • This means there must be two pairs of equal numbers.
  • Strategy: Sort the four numbers. Let sorted values be $s_1, s_2, s_3, s_4$.
  • For a rectangle, the two smallest must be equal ($s_1 = s_2$) and the two largest must be equal ($s_3 = s_4$).
  • (Note: A square is also a rectangle, $s_1=s_2=s_3=s_4$ satisfies this).

1.1 Dry Run

Example: $1, 2, 2, 1$

  • Sort: 1, 1, 2, 2.
  • $1=1$ and $2=2$. YES.

Example: $1, 2, 3, 4$

  • Sort: 1, 2, 3, 4.
  • $1 \ne 2$. NO.

1.2 Pseudocode Solution

READ T
FOR i FROM 1 TO T:
    READ a, b, c, d
    List L = [a, b, c, d]
    SORT(L)
    
    IF L[0] == L[1] AND L[2] == L[3]:
        PRINT "YES"
    ELSE:
        PRINT "NO"
  • Time Complexity: $O(1)$ (Sorting 4 elements).
  • Space Complexity: $O(1)$.
