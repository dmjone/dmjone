Average of Three

View Problem Online

1. Problem Statement & Logic

Problem: Given an integer $X$. Find three distinct integers $A, B, C$ such that their average is $X$. i.e., $(A+B+C)/3 = X$.

Logic:

  • We need $A+B+C = 3X$.
  • We can pick arbitrary values for $A$ and $B$, then solve for $C$.
  • Let $A = X-1$, $B = X$, $C = X+1$.
  • Sum: $(X-1) + X + (X+1) = 3X$. Average is $X$.
  • These are distinct integers for any $X$. Wait, they are distinct if $X$ is large enough or specific conditions?
  • Actually, easiest distinct set: $A=1, B=2, C=3X-3$. But need to check if $C$ distinct from $A, B$.
  • The set $\{X-1, X, X+1\}$ always works and provides distinct integers.
  • Output: $X-1, X, X+1$.

1.1 Dry Run

Example: $X=3$

  • $A=2, B=3, C=4$. Sum=9. Average=3. Distinct? Yes. Output: 2 3 4.

1.2 Pseudocode Solution

READ T
FOR i FROM 1 TO T:
    READ X
    PRINT (X-1) + " " + X + " " + (X+1)
  • Time Complexity: $O(1)$.
  • Space Complexity: $O(1)$.
