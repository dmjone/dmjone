1. Problem Statement & Logic

Problem: Kill $N$ enemies with health $H_i$. Mode 1: Damage $X$ to one enemy/sec. Mode 2: Damage $1$ to all enemies/sec. Min time?

Logic:

  • Calculate time for Mode 1:
    • For enemy $i$, time $t_i = \lceil H_i / X \rceil$.
    • Total time $T_1 = \sum t_i$.
  • Calculate time for Mode 2:
    • Damage 1 to all per second. Time is limited by max health.
    • $T_2 = \max(H)$.
  • Result: $\min(T_1, T_2)$.

1.1 Dry Run

Example: $N=3, X=4, H=[5, 10, 15]$

  • Mode 1:
    • 5: $\lceil 5/4 \rceil = 2$s.
    • 10: $\lceil 10/4 \rceil = 3$s.
    • 15: $\lceil 15/4 \rceil = 4$s.
    • Sum: $2+3+4=9$s.
  • Mode 2: Max(5, 10, 15) = 15s.
  • Min(9, 15) = 9s. Output: 9.

1.2 Pseudocode Solution


READ T
FOR i FROM 1 TO T:
    READ N, X
    READ H (array)
    
    time_mode1 = 0
    time_mode2 = 0
    
    FOR h in H:
        # Ceiling division for Mode 1
        curr_time = (h + X - 1) // X
        time_mode1 = time_mode1 + curr_time
        
        # Max for Mode 2
        IF h > time_mode2:
            time_mode2 = h
            
    PRINT MIN(time_mode1, time_mode2)
  • Time Complexity: $O(N)$.
  • Space Complexity: $O(1)$.
Why is the time for Mode 1 $\sum \lceil H_i / X \rceil$?

A: Mode 1 requires focusing on one enemy at a time. Each enemy takes a discrete number of seconds ($\lceil H_i/X \rceil$) to defeat. The total time is simply the sum of these individual times.
