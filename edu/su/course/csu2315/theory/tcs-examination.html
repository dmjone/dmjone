1. Problem Statement & Logic

Problem: Compare ranks of Dragon ($D$) and Sloth ($S$) based on scores in DSA, TOC, DM.
Rules:

  1. Higher Total Score wins.
  2. If Total tied, higher DSA wins.
  3. If DSA tied, higher TOC wins.
  4. Else, Tie.

Logic:

  • Calculate totals: $D_{total} = D_{DSA} + D_{TOC} + D_{DM}$. Similarity for $S$.
  • Compare $D_{total}$ vs $S_{total}$. If unequal, winner is clear.
  • If totals equal, compare $D_{DSA}$ vs $S_{DSA}$.
  • If DSA equal, compare $D_{TOC}$ vs $S_{TOC}$.
  • If TOC equal, then it's a TIE.

1.1 Dry Run

Example: D: [10, 20, 30] (Total 60), S: [20, 20, 20] (Total 60)

  • Totals equal. Compare DSA.
  • $D_{DSA}=10, S_{DSA}=20$. Sloth wins. Output: "Sloth".

1.2 Pseudocode Solution

READ T
FOR i FROM 1 TO T:
    READ D_DSA, D_TOC, D_DM
    READ S_DSA, S_TOC, S_DM
    
    D_total = D_DSA + D_TOC + D_DM
    S_total = S_DSA + S_TOC + S_DM
    
    IF D_total > S_total:
        PRINT "Dragon"
    ELSE IF S_total > D_total:
        PRINT "Sloth"
    ELSE:
        IF D_DSA > S_DSA:
            PRINT "Dragon"
        ELSE IF S_DSA > D_DSA:
            PRINT "Sloth"
        ELSE:
            IF D_TOC > S_TOC:
                PRINT "Dragon"
            ELSE IF S_TOC > D_TOC:
                PRINT "Sloth"
            ELSE:
                PRINT "Tie"
  • Time Complexity: $O(1)$.
  • Space Complexity: $O(1)$.
