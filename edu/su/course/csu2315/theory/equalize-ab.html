1. Problem Statement & Logic

Problem: Given $A, B, X$. Can we make $A$ equal to $B$ by operations: $A \leftarrow A+X, B \leftarrow B-X$ OR $A \leftarrow A-X, B \leftarrow B+X$?

Logic:

  • Let total sum $S = A + B$. In any operation, $A$ changes by $\pm X$ and $B$ changes by $\mp X$.
  • Note that $(A+X) + (B-X) = A + B$. The sum $A+B$ remains invariant (constant).
  • If we want $A$ and $B$ to become equal, final state must be $A_{final} = B_{final}$.
  • So $2 \times A_{final} = S \implies A_{final} = (A+B)/2$.
  • Condition 1: $A+B$ must be even (divisible by 2). If odd, cannot be equal integers.
  • Condition 2: We need to reach $(A+B)/2$ from $A$ using steps of size $X$.
  • Difference required = $|A - A_{final}| = |A - (A+B)/2| = |(2A - A - B)/2| = |(A-B)/2|$.
  • This difference must be a multiple of $X$.
  • So check: $((A-B)/2) \% X == 0$.
  • Or simply: Is $(A-B)$ divisible by $2X$?
  • Wait, each step changes difference by $2X$ (A gets $+X$, B gets $-X$, diff changes by $2X$).
  • Initial Diff = $|A-B|$. We want final Diff = 0.
  • So $|A-B|$ must be divisible by $2X$.

1.1 Dry Run

Example: $A=5, B=7, X=1$

  • Diff $|5-7| = 2$.
  • $2X = 2$. $2 \% 2 == 0$. Yes.
  • Operation: $5 \to 6, 7 \to 6$. Max equal. YES.

Example: $A=5, B=7, X=2$

  • Diff 2. $2X = 4$. $2 \% 4 \neq 0$. NO.

1.2 Pseudocode Solution


READ T
FOR i FROM 1 TO T:
    READ A, B, X
    
    diff = ABS(A - B)
    step = 2 * X
    
    IF diff % step == 0:
        PRINT "YES"
    ELSE:
        PRINT "NO"
  • Time Complexity: $O(1)$.
  • Space Complexity: $O(1)$.
Why must the difference $|A-B|$ be divisible by $2X$?

A: Because each operation changes $A$ by $\pm X$ and $B$ by $\mp X$, changing the difference $(A-B)$ by exactly $\pm 2X$. To reach a difference of 0, the initial difference must be a multiple of $2X$.
