Adjacent Sum Parity

2025, Dec 15
View Problem Online

1. Problem Statement & Logic

Problem: Determine if there exists a binary array $A$ of size $N$ such that a given binary array $B$ represents the parity ($sum \% 2$) of adjacent elements of $A$. Specifically, $B_i = (A_i + A_{i+1}) \% 2$ for $1 \le i < N$, and $B_N=(A_N + A_1) \% 2$.

Logic: Summing all elements of $B$ gives the sum of all adjacent pairs in $A$. Since each $A_i$ appears exactly twice in this sum (once as $A_i + A_{i+1}$ and once as $A_{i-1} + A_i$), the total sum is $2 \cdot \sum A_i$. In modulo 2 arithmetic, any number multiplied by 2 is 0 (even). Therefore, a solution exists if and only if the sum of elements in $B$ is even.

  • Input/Output Relation: If $\sum B_i$ is even $\rightarrow$ Output "YES". If $\sum B_i$ is odd $\rightarrow$ Output "NO".

1.1 Dry Run

Consider $N=2$.

  • Case 1 (B = [1, 1]): Sum = $1+1=2$ (Even). Output: YES.
    • Verification: Let $A=[1, 0]$. $B_1 = (1+0)\%2 = 1$, $B_2 = (0+1)\%2 = 1$. Matches.
  • Case 2 (B = [1, 0]): Sum = $1+0=1$ (Odd). Output: NO.
    • Verification: Impossible. $A_1+A_2$ is odd and $A_2+A_1$ is even, which is a contradiction.

1.2 Pseudocode Solution

# Read number of test cases
READ T

FOR i FROM 1 TO T:
    # Read size of array
    READ N
    
    # Read binary array B
    READ B of size N
    
    # Calculate sum of elements in B
    SUM = 0
    FOR j FROM 0 TO N-1:
        SUM = SUM + B[j]
    
    # Check if sum is even or odd
    IF SUM % 2 == 0:
        PRINT "YES"
    ELSE:
        PRINT "NO"
  • Time Complexity: $O(N)$ - we iterate through the array once.
  • Space Complexity: $O(1)$ - if we compute sum on the fly.
