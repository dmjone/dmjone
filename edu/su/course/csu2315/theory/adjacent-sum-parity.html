Adjacent Sum Parity

2025, Dec 15
View Problem Online

1. Problem Statement & Logic

Problem: Given a binary array $B$ of size $N$, determine whether there exists an integer array $A$ such that:

  • $B_i = (A_i + A_{i+1}) \% 2$ for $1 \le i < N$
  • $B_N = (A_N + A_1) \% 2$

Logic:

  • Only the parity (even or odd) of elements of $A$ matters.
  • Let us add all values of $B$:

    $$\sum_{i=1}^{N} B_i = \sum_{i=1}^{N} (A_i + A_{i+1}) \% 2$$

  • Each element $A_i$ appears in exactly two adjacent sums:
    • Once in $(A_{i-1} + A_i)$
    • Once in $(A_i + A_{i+1})$
  • So, when all adjacent sums are added together, we get:

    $$\sum (A_i + A_{i+1}) = 2 \cdot \sum A_i$$

  • Since this value is multiplied by 2, it is always even.
  • Therefore, the sum of all values in $B$ must be even.
  • If $\sum B_i$ is even, such an array $A$ can be constructed.
  • If $\sum B_i$ is odd, it is impossible.

  • Final Condition:
    • If $\sum B_i$ is even → print YES
    • If $\sum B_i$ is odd → print NO

1.1 Dry Run

Example 1: $N = 2$, $B = [0, 0]$

  • Sum of B = 0 + 0 = 0 (even)
  • Valid array exists
  • Output = YES

Example 2: $N = 2$, $B = [1, 0]$

  • Sum of B = 1 + 0 = 1 (odd)
  • Impossible to form array A
  • Output = NO

Example 3: $N = 4$, $B = [1, 0, 1, 0]$

  • Sum of B = 1 + 0 + 1 + 0 = 2 (even)
  • Valid array exists
  • Output = YES

1.2 Pseudocode Solution


READ T
FOR each test case:
    READ N
    READ array B of size N

    sum = 0
    FOR i FROM 1 TO N:
        sum = sum + B[i]

    IF sum % 2 == 0:
        PRINT "YES"
    ELSE:
        PRINT "NO"
  • Time Complexity: $O(N)$ per test case.
  • Space Complexity: $O(1)$.
