1. Problem Statement & Logic

Problem: Given $N$ bags with $A_i$ coins. Max capacity of a bag is $X$. You can pay cost $C$ to fill any bag to $X$. Maximize: $(\sum \text{Final Coins}) - (\text{Total Cost})$.

Logic:

  • For each bag $i$, we have two choices:
    1. Leave it as $A_i$. Contribution to value: $A_i$. (Cost 0).
    2. Fill it to $X$. Contribution to value: $X$. Cost: $C$. Net contribution: $X - C$.
  • We should choose option 2 if $X - C > A_i$. Otherwise, leave it.
  • Iterate through all bags. If $A_i < X - C$, replace $A_i$ with $X$ and subtract $C$ from profit (or simply sum max contribution).
  • Actually easiest way: Initialize `total_val` = sum of initial $A_i$. Improve it.
  • Gain from operation on bag $i$: $(X - A_i) - C$. If this gain $> 0$, do it.
  • Or simply: Contribution from bag $i = \max(A_i, X - C)$. Sum these up.

1.1 Dry Run

Example: $N=3, X=5, C=1, A=[1, 2, 3]$

  • $X-C = 5-1 = 4$.
  • Bag 1 ($1$): $\max(1, 4) = 4$. (Operation used).
  • Bag 2 ($2$): $\max(2, 4) = 4$. (Operation used).
  • Bag 3 ($3$): $\max(3, 4) = 4$. (Operation used).
  • Total: $4+4+4 = 12$.

Example: $N=3, X=5, C=3, A=[1, 2, 3]$

  • $X-C = 5-3 = 2$.
  • Bag 1 ($1$): $\max(1, 2) = 2$.
  • Bag 2 ($2$): $\max(2, 2) = 2$.
  • Bag 3 ($3$): $\max(3, 2) = 3$.
  • Total: $2+2+3 = 7$.

1.2 Pseudocode Solution

READ T
FOR i FROM 1 TO T:
    READ N, X, C
    READ A (array)
    
    total_val = 0
    target_val = X - C
    
    FOR val in A:
        IF val < target_val:
            total_val = total_val + target_val
        ELSE:
            total_val = total_val + val
            
    PRINT total_val
  • Time Complexity: $O(N)$.
  • Space Complexity: $O(1)$.
