1. Problem Statement & Logic

Problem: Given a string $S$ and costs for each of the 26 alphabets. Buy missing characters to make $S$ a pangram (contains all 'a'-'z'). Minimize cost.

Logic:

  • Mark which characters are present in $S$.
  • Iterate from 'a' to 'z'.
  • If a character is NOT present, add its cost to the total.
  • Print total cost.

1.1 Dry Run

Example: Costs: 1, 2, 3... (rest 0). $S=$ "abc".

  • Present: a, b, c.
  • Missing: d, e, f... z.
  • Sum costs of all missing chars.

1.2 Pseudocode Solution

READ T
FOR i FROM 1 TO T:
    READ Costs (array of 26 integers)
    READ S
    
    # Mark present characters
    Present = [FALSE] * 26
    FOR char in S:
        index = ASCII(char) - ASCII('a')
        Present[index] = TRUE
        
    total_cost = 0
    FOR j FROM 0 TO 25:
        IF Present[j] == FALSE:
            total_cost = total_cost + Costs[j]
            
    PRINT total_cost
  • Time Complexity: $O(N)$ (to scan string) + $O(26)$.
  • Space Complexity: $O(1)$ (freq array size 26).
