Retrieve the Array

1. Problem Statement & Logic

Problem: Unknown array $A$. Given array $B$ where $B_i = \text{PrefixSum}[i] + \text{SuffixSum}[i]$. Recover $A$.

Logic:

  • $B_i = (A_1 + \dots + A_i) + (A_i + \dots + A_N)$.
  • $B_i = \text{TotalSum}(A) + A_i$.
  • So $\sum B_i = \sum (\text{TotalSum}(A) + A_i) = N \times \text{TotalSum}(A) + \text{TotalSum}(A) = (N+1) \times \text{TotalSum}(A)$.
  • Let $S_A = \text{TotalSum}(A)$. Then $\sum B = (N+1) S_A$.
  • $S_A = \frac{\sum B}{N+1}$.
  • Then $A_i = B_i - S_A$.

1.1 Dry Run

Example: $A=[1, 2]$. $S_A=3$.

  • $B_1 = (1) + (1+2) = 4$. $B_2 = (1+2) + (2) = 5$.
  • Sum B = 9. $N=2$. $S_A = 9 / 3 = 3$.
  • $A_1 = 4 - 3 = 1$. $A_2 = 5 - 3 = 2$. Correct.

1.2 Pseudocode Solution


READ T
FOR i FROM 1 TO T:
    READ N
    READ B (array)
    
    sum_b = 0
    FOR x in B:
        sum_b = sum_b + x
        
    s_a = sum_b / (N + 1)
    
    FOR x in B:
        PRINT (x - s_a) + " "
    PRINT NEWLINE
  • Time Complexity: $O(N)$.
  • Space Complexity: $O(1)$.
How do we derive $S_A = \sum B / (N+1)$?

A: Since each $B_i = S_A + A_i$, summing all $B_i$ gives $\sum S_A + \sum A_i$. Since there are $N$ terms, this becomes $N \times S_A + S_A = (N+1)S_A$. Thus, $S_A = \text{Sum}(B) / (N+1)$.
