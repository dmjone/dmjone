Break the Stick

1. Problem Statement & Logic

Problem: Chef has a stick of length $N$. He can break it into parts such that all parts have the same parity (all even or all odd). He can repeat this process. Determine if he can obtain a stick of length exactly $X$.

Logic: The core constraint is the parity of the parts.

  • If $N$ is Odd: We can only break it into Odd parts (sum of evens is even, so odd sum requires odd parts). Thus, from an Odd stick, we can only ever get Odd sticks. So, if $X$ is Even, it's impossible. If $X$ is Odd, it's possible (just break off $X$ and keep remainder $N-X$, which is even... wait, we need parts to have same parity. $N=11, X=3$. $N-X=8$. Can we split 11 into odds involving 3? Yes, $11 = 3 + 3 + 5$. All odd. So if $X$ is Odd, Yes).
  • If $N$ is Even:
    • We can break it into Evens (e.g. $N = X + (N-X)$ where $X$ is Even).
    • We can break it into Odds (e.g. $N = X + (N-X)$ where $X$ is Odd). Note that if $X$ is odd, $N-X$ is odd (Even - Odd = Odd). So the two parts $X$ and $N-X$ are both odd. Valid.
    So if $N$ is Even, we can get any $X$ (Even or Odd).
Conclusion: If ($N \% 2 == 0$) OR ($X \% 2 != 0$), the answer is "YES". Otherwise "NO".

1.1 Dry Run

Case 1: $N=10, X=4$

  • $N$ is Even. We can split $10 \to 4 + 6$. Both even. Valid.
  • Output: YES.

Case 2: $N=11, X=4$

  • $N$ is Odd. Can only break into odd parts. $4$ is Even. Impossible.
  • Output: NO.

Case 3: $N=11, X=5$

  • $X$ is Odd. $N$ is Odd.
  • We can write $11 = 5 + 3 + 3$. All odd. Valid.
  • Output: YES.

1.2 Pseudocode Solution

READ T
FOR i FROM 1 TO T:
    READ N, X
    
    # If N is Even, we can always make X (Odd or Even)
    IF N % 2 == 0:
        PRINT "YES"
        
    # If N is Odd, X must be Odd
    ELSE:
        IF X % 2 != 0:
            PRINT "YES"
        ELSE:
            PRINT "NO"
  • Time Complexity: $O(1)$ per test case.
  • Space Complexity: $O(1)$.
