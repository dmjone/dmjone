1. Problem Statement & Logic

Problem: Given $N$ tasks with times $T_i$ and data $D_i$. Available "free data" $K$. Consume free data to pay for $D_i$. If $D_i$ fully paid, time $T_i$ is 0. If $D_i$ partially paid, time remains. Wait.
Usually "Download file" involves paying for data? Or just "Chef downloads files".
Common variant: For each file $i$, time $T_i$ and data cost $K_i$. We have total $K$ free data. We want to minimize total time?
Usually: If used data $\ge K_i$, time is 0. Else time is $T_i \times (K_i - \text{used})$. No.
Likely Problem: "Download File" (DWNLD).
For each minute $T_i$, Chef downloads at rate $D_i$. First $K$ data is free. Calculate cost for remaining?
Check DWNLD problem:
Chef pays 1 dollar per 1 data. First $K$ data is free.
Input: $N$ intervals. $i$-th: Duration $T_i$, Rate $D_i$.
Total Data = $\sum T_i \times D_i$.
Billable Data = $\max(0, \text{Total Data} - K)$? No, usually "First K units of data are free".
Wait, is it "First K minutes free" or "First K data free"? Problem usually "First K units of data".
Logic: Calculate total cost. Pay for all data needed. Subtract cost of first $K$.
Actually, if $K$ covers a partial interval?
Iterate intervals. For current interval $i$ with $T_i, D_i$:

  • Data in this interval = $T_i \times D_i$.
  • If $K \ge \text{Data}$: Free. $K = K - \text{Data}$. Cost 0.
  • If $K < \text{Data}$: Free part is $K$. Paid part is $\text{Data} - K$. Cost +=$(\text{Data} - K)$. $K=0$.

1.1 Dry Run

Example: $K=10$. Inter 1: 2 min, 2 rate $\to$ 4 data. Inter 2: 2 min, 4 rate $\to$ 8 data.

  • Interval 1: Data 4. $K=10 \ge 4$. Free. $K=6$.
  • Interval 2: Data 8. $K=6 < 8$. Free 6. Paid 2. Cost 2.
  • Output: 2.

1.2 Pseudocode Solution

READ TC
FOR i FROM 1 TO TC:
    READ N, K
    total_cost = 0
    
    FOR j FROM 1 TO N:
        READ T, D
        interval_data = T * D
        
        IF K >= interval_data:
            K = K - interval_data
        ELSE:
            paid_data = interval_data - K
            total_cost = total_cost + paid_data
            K = 0
            
    PRINT total_cost
  • Time Complexity: $O(N)$.
  • Space Complexity: $O(1)$.
