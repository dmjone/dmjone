Automaton Designer - Self Help

How to use the Automaton Designer

Click outside to close and start designing!
  • Add a state: Double-click on the canvas.
  • Add an arrow: Shift + drag on the canvas.
  • Move something: Drag it around.
  • Delete something: Click it and press the delete key.
  • Make accept state: Double-click on an existing state.
  • Type numeric subscript: Put an underscore before the number (e.g., "S_0").
  • Type Greek letter: Put a backslash before it (e.g., "\beta").

Available Symbols:

You can use the following symbols by typing a backslash (\) followed by the symbol name:

  • \Alpha, \Beta, \Gamma, \Delta, \Epsilon, \Zeta, \Eta, \Theta, \Iota, \Kappa, \Lambda, \Mu, \Nu, \Xi, \Omicron, \Pi, \Rho, \Sigma, \Tau, \Upsilon, \Phi, \Chi, \Psi, \Omega, \rightarrow, \leftarrow, \emptyset, \eps

This interactive tool leverages HTML5 and JavaScript, utilizing the canvas element for dynamic and creative content creation.
