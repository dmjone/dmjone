1. Digital Image Processing (Very Simple Understanding)

What is an Image?

An image is basically a function: $f(x,y)$

$x, y$ → position of pixel

$f(x,y)$ → brightness or intensity at that point

Example: A photo on your phone — each small dot = pixel. Image = collection of pixels. Each pixel stores an intensity value.

2. What is Digital Image Processing (DIP)?

Definition

Digital Image Processing means using computers to process and analyze digital images.

Examples: remove noise, increase brightness, detect objects, medical scanning.

3. Pixel

A pixel is the smallest unit of an image.

Example: A photo: 1000 × 1000 pixels → Total pixels = 1,000,000

Each pixel has: location and intensity value.

4. Levels of Image Processing (VERY IMPORTANT FOR EXAM)

There are three levels:

  • Low-Level Processing

    Input → Image, Output → Image. Examples: noise removal, image sharpening, contrast enhancement. Example: Blurry image → Clear image.

  • Mid-Level Processing

    Input → Image, Output → Features. Examples: edge detection, object segmentation. Example: Detect car boundaries in a photo.

  • High-Level Processing

    Input → Features, Output → Understanding. Examples: scene understanding, object recognition, autonomous cars. Example: Computer understands "THIS IS A CAR".

5. History of Digital Image Processing

1920s — Newspapers transmitted images using early systems; transmission time reduced drastically.

1960s — Space race: space probes sent distorted images; scientists used computers to restore images.

1970s — Medical imaging revolution: CT scan, MRI; computers reconstruct body images.

6. Sources of Digital Images

Images can come from different parts of the electromagnetic spectrum and other energy sources:

  • Gamma Rays — used for nuclear medicine and astronomy.
  • X-Rays — medical imaging, airport security (bone X-ray).
  • Ultraviolet — astronomy and scientific imaging.
  • Visible Light — normal cameras use visible spectrum (≈400 nm–700 nm). Colors: Red, Green, Blue.
  • Infrared — detects heat (thermal cameras, night vision).
  • Microwaves — radar imaging, satellite imaging.
  • Radio Waves — MRI scans.
  • Ultrasound — uses sound waves (e.g., pregnancy scans).

7. Image Processing Pipeline (EXTREMELY IMPORTANT)

Every image processing system follows these steps:

  1. Image Acquisition

    Capture image using camera or sensor, then convert to a digital image.

  2. Enhancement

    Improve image quality — brightness correction, sharpening, noise removal. Purpose: make image look better (subjective).

  3. Restoration

    Recover original image using mathematics — e.g., remove motion blur. Difference: restoration is mathematical, enhancement is subjective.

  4. Morphological Processing

    Deals with shape of objects — boundary extraction, skeleton detection.

  5. Segmentation (MOST IMPORTANT)

    Divide image into parts (e.g., car, road, sky). This step is the most difficult.

  6. Recognition

    Final step: computer identifies objects (e.g., "this is a tumor", "this is a vehicle").

Parts of the eye: cornea, iris, lens, retina.

Retina contains receptors:

  • Rods: 75–150 million — detect low light, no color detection.

10. Image Formation Model (VERY IMPORTANT)

Image formation equation:

$$f(x,y) = i(x,y) \times r(x,y)$$

Where:

  • f(x,y): image intensity
  • i(x,y): illumination — amount of light hitting the object
  • r(x,y): reflectance — amount of light reflected by the object

Reflectance range: $0 \\le r(x,y) \\le 1$ (white wall → high reflectance; black cloth → low reflectance).

11. Image Sampling and Quantization

Converts continuous image → digital image. Two steps:

  • Sampling: convert spatial coordinates to pixels (e.g., image = 512 × 512 pixels). Determines spatial resolution.
  • Quantization: convert intensity values (e.g., 8-bit image → $2^8 = 256$ intensity levels, range 0 → 255).

12. Spatial Resolution

Definition: ability to distinguish small details. Measured by pixels per unit area. Example: 4K image → high spatial resolution.

13. Intensity Resolution

Definition: ability to distinguish small brightness differences. Measured by number of bits. Example: 8-bit → 256 levels; 10-bit → 1024 levels.

14. Effects of Low Resolution

Low Spatial Resolution → jagged edges, blurry shapes.

Low Intensity Resolution → false contouring, banding.
