1. Why Morphology?

Earlier filters work on pixels. Morphology works on shapes.

  • Foundation: set theory
  • Probe: structuring element (SE), a small binary template (3×3 square, cross, disk)
  • Origin of SE placed on each foreground pixel

2. Erosion

Shrinks objects.

Pixel survives only if entire SE fits within foreground.

  • Boundaries recede inward
  • Small objects disappear
  • Thin connections break

Use: remove small noise.

3. Dilation

Grows objects (dual of erosion).

Stamp the SE at every foreground pixel.

  • Boundaries expand
  • Small gaps fill in
  • Nearby objects merge

Use: bridge broken contours, fill small holes.

4. Opening and Closing

Combine erosion and dilation to clean shapes without changing size.

  • Opening = Erosion → Dilation. Removes outside noise. Smooths outer contour.
  • Closing = Dilation → Erosion. Fills inside holes. Smooths inner contour.

Noise filter: Opening then closing = remove external + internal noise.

Implementation
opened = cv2.morphologyEx(img, cv2.MORPH_OPEN, SE)
closed = cv2.morphologyEx(img, cv2.MORPH_CLOSE, SE)

5. What is Segmentation?

Divide the image into meaningful regions.

Two strategies:

  • Discontinuity-based: detect edges (Sobel, Laplacian, Canny)
  • Similarity-based: group similar pixels (thresholding, region growing)

6. Edge Models

  • Step: idealized 1-pixel transition
  • Ramp: realistic gradual transition (blur, noise)
  • Roof: thin line (rise → plateau → fall)

Detection:

  • 1st derivative → peak at edge midpoint
  • 2nd derivative → zero-crossing at edge midpoint

7. Thresholding

$$ g(x,y) = \begin{cases} 1 & \text{if } f(x,y) \geq T \\ 0 & \text{otherwise} \end{cases} $$

Best when histogram is bimodal.

7.1 Global vs. Variable

  • Global: one $T$ for whole image (uniform illumination)
  • Variable / Adaptive: $T$ varies spatially (uneven lighting)

8. Otsu's Method

Pick $T$ that maximizes between-class variance:

$$ \sigma_B^2(T) = \omega_1 \omega_2 (\mu_1 - \mu_2)^2 $$

8.1 Algorithm

  • Compute normalized histogram
  • For each $T$: compute $\omega_1, \omega_2, \mu_1, \mu_2, \sigma_B^2$
  • Pick $T^*$ maximizing $\sigma_B^2$

Key win: fully automatic, no parameters.

Implementation
T, binary = cv2.threshold(img, 0, 255,
    cv2.THRESH_BINARY + cv2.THRESH_OTSU)

9. Region Growing

Groups pixels by similarity (not edges).

  • Pick seeds inside regions of interest
  • Define similarity criterion (e.g., $|f - \mu_{\text{region}}| < T$)
  • Grow by adding neighbors that satisfy criterion
  • Stop when no neighbors qualify

Strength: connected regions by construction. Weakness: sensitive to seeds + threshold.

10. Decision Engine (Exam Gold)

  • Remove small noise? → Erosion or Opening
  • Bridge broken edges? → Dilation or Closing
  • Clean both? → Opening + Closing
  • Bimodal histogram, uniform light? → Global thresholding (Otsu)
  • Uneven illumination? → Variable thresholding
  • Known seed inside object? → Region growing
  • Find boundaries? → Sobel/Laplacian (discontinuity-based)

Master pair: erosion ↔ dilation, opening ↔ closing, edge ↔ region.
