1. Neighbors of a Pixel

Concept

In an image every point is a pixel. Each pixel has neighboring pixels around it.

Example structure:


            p1  p2  p3
            p4  P   p5
            p6  p7  p8

P is the center pixel and the surrounding pixels are its neighbors.

4-Neighborhood

Formula:

$$N_4(p)=\{(x-1,y),(x+1,y),(x,y-1),(x,y+1)\}$$

Meaning: Only pixels touching vertically or horizontally are neighbors.


                  (x,y-1)
            
            (x-1,y)  P  (x+1,y)
            
                  (x,y+1)

8-Neighborhood

Formula:

$$N_8(p)=\{(x-1,y),(x+1,y),(x,y-1),(x,y+1),(x-1,y-1),(x+1,y-1),(x-1,y+1),(x+1,y+1)\}$$

Meaning: All eight surrounding pixels are neighbors.


            p1  p2  p3
            p4  P   p5
            p6  p7  p8

2. Paths Between Pixels

Concept

A path means traveling from one pixel to another through connected pixels.

Example: Start → End through neighbors.

Formula

A path is represented as:

$$(x_0,y_0),(x_1,y_1),...,(x_n,y_n)$$

Where:

  • (x0,y0): starting pixel
  • (xn,yn): ending pixel

Each pixel must be a neighbor of the previous pixel.

Meaning

Movement occurs through connected pixels.

Example path:

A → B → C → D

Length of path = number of steps.

Important Terms

  • Connected Component: Pixels connected together forming a group.
  • Region: Connected pixels representing the same object.
  • Foreground: Object pixels.
  • Background: Remaining pixels in the image.

3. Distance Between Pixels

Distance measures how far two pixels are from each other.

Euclidean Distance

Formula:

$$D_e(p,q)=\sqrt{(x-u)^2+(y-v)^2}$$

Meaning: Standard geometric distance between two points.

Example:

Pixel1: (2,3)

Pixel2: (5,7)

$$\sqrt{(2-5)^2+(3-7)^2}=\sqrt{9+16}=\sqrt{25}=5$$

City Block Distance

Formula:

$$D_4(p,q)=|x-u|+|y-v|$$

Meaning: Distance measured using only horizontal and vertical movement.

Example:

|2-5| + |3-7| = 3 + 4 = 7

Chessboard Distance

Formula:

$$D_8(p,q)=max(|x-u|,|y-v|)$$

Meaning: Distance based on movement similar to a chess king (diagonal allowed).

Example:

(2,3) → (5,7)

max(3,4) = 4

4. Element-wise vs Matrix Operations

Element-wise Operation

Each pixel is multiplied with the corresponding pixel.

Example:


            Image A
            
            1 2
            3 4
            
            Image B
            
            5 6
            7 8

Result:


            1×5   2×6
            3×7   4×8
            
            5 12
            21 32

Matrix Multiplication

Uses linear algebra rules.

More complex and usually not used frequently in pixel operations.

5. Linear vs Non-Linear Operations

Operator form:

$$H[f(x,y)]=g(x,y)$$

Meaning: Input image → operator → output image.

Linear Operator

Must satisfy two conditions.

Homogeneity

$$H[af]=aH[f]$$

If input doubles, output doubles.

Additivity

$$H[f_1+f_2]=H[f_1]+H[f_2]$$

Processing images separately and adding results produces the same result.

Example: Mean filter.

Non-Linear Operator

Does not satisfy the above conditions.

Example: Max filter.

6. Arithmetic Operations

Addition

Formula:

$$s(x,y)=f(x,y)+g(x,y)$$

Meaning: Add corresponding pixel values.

Example: 100 + 50 = 150

Subtraction

Formula:

$$d(x,y)=f(x,y)-g(x,y)$$

Used for detecting changes between images.

Multiplication

Formula:

$$p(x,y)=f(x,y)\times g(x,y)$$

Used for shading correction.

Division

Formula:

$$v(x,y)=\frac{f(x,y)}{g(x,y)}$$

Used to remove lighting effects.

7. Noise Reduction Using Averaging

Noise model:

$$g(x,y)=f(x,y)+\eta(x,y)$$

Where:

  • g: noisy image
  • f: original image
  • η: noise

Noise reduction using averaging:

$$\bar{g}(x,y)=\frac{1}{K}\sum g_i(x,y)$$

Meaning: Average multiple captured images so noise cancels out.

8. Set Operations

Images can be treated like mathematical sets.

Complement

Formula:

$$s=K-r$$

Where:

  • K: maximum intensity (255 for 8-bit image)
  • r: original intensity

Example:

Pixel value = 50

255 − 50 = 205

This produces a negative image.

9. Logical Operations

Applied on binary images.

  • 1 → object
  • 0 → background

Operations include:

  • AND
  • OR
  • NOT
  • XOR

Example:

1 AND 1 = 1

1 AND 0 = 0

Used in object detection.

10. Spatial Operations

Operations applied directly to image pixels.

Types:

  • Single pixel operations
  • Neighborhood operations
  • Geometric transformations

11. Single Pixel Operations

Formula:

$$S=T(z)$$

Where:

  • z: original pixel value
  • S: transformed pixel value
  • T: transformation function

Examples:

  • Negative image
  • Contrast stretching
  • Thresholding

12. Neighborhood Operations

Uses surrounding pixels to modify a pixel.

Example: Average filter.

Formula:

$$g(x,y)=\frac{1}{mn}\sum f(r,c)$$

Meaning: Average the values of neighboring pixels.

Applications:

  • Noise reduction
  • Image smoothing
  • Blurring

13. Geometric Transformations

Transforms the shape or position of an image.

Examples:

  • Translation
  • Rotation
  • Scaling
  • Shearing

Interpolation

Required after geometric transformation to estimate new pixel values.

Types:

  • Nearest Neighbor
  • Bilinear
  • Bicubic

14. Spatial vs Transform Domain

Spatial Domain

Processing done directly on pixel values.

Example: brightness adjustment.

Transform Domain

Image is converted to frequency domain.

Examples:

  • Fourier Transform
  • Discrete Cosine Transform (DCT)
  • Wavelets

Used for compression and noise removal.

15. Image Transform

General transform formula:

$$T(u,v)=\sum f(x,y)r(x,y,u,v)$$

Meaning: Converts image from spatial domain to transform domain.

Inverse transform is used to reconstruct the image.

Example: Fourier transform used for removing interference noise.

16. Probability of Intensity Levels

Pixel intensity can be treated as a random variable.

Formula:

$$p(z_k)=\frac{n_k}{MN}$$

Where:

  • nk: number of pixels with intensity zk
  • MN: total number of pixels

Example:

Total pixels = 100

Pixels with intensity 50 = 10

Probability = 10 / 100 = 0.1

17. Mean Intensity

Represents the average brightness of the image.

Formula:

$$\mu=\sum z_i p(z_i)$$

18. Variance

Measures the variation in brightness values.

Formula:

$$\sigma^2=\sum (z_i-\mu)^2 p(z_i)$$
