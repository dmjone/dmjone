Digital Image Processing

2026, March 8
View Summary By Jatin

1. What is a Digital Image?

  • A 2D function f(x, y) where x, y = spatial coordinates, f = intensity (brightness)
  • When x, y, and f are all finite and discrete, it is a digital image
  • Pixel (picture element): smallest unit; has a location (x, y) and an intensity value
  • Image of size M × N = M rows × N columns = M × N pixels
  • DIP: processing digital images using computers

2. Levels of Image Processing

  • Low-Level: Image → Image. Pixel-level cleanup. E.g., noise reduction, contrast enhancement, sharpening
  • Mid-Level: Image → Attributes. Extracts edges, regions, features. E.g., segmentation, edge detection
  • High-Level: Attributes → Understanding. Interprets scene meaning. E.g., scene recognition, autonomous navigation

3. Origins of Digital Image Processing

  • 1920s: Bartlane cable system transmitted images across Atlantic; 5 gray levels, under 3 hours
  • 1960s: Space probe images enhanced using computers; birth of modern DIP
  • 1970s: CT scanning reconstructed internal body structures from projections; proved machines can reveal what humans cannot perceive

4. Sources of Digital Images

Principal source: Electromagnetic (EM) spectrum. Shorter wavelength = higher energy = deeper penetration.

4.1 EM Spectrum

Band Application

UltravioletCrop disease detection

4.2 Non-EM Sources

  • Ultrasound: Sound waves reflecting off tissue; pregnancy scans
  • Electron Microscopy: Nanometer-scale imaging
  • Computer-Generated: Synthetic images (3D models, fractals)

5. Fundamental Pipeline (AEMSR)

  • Acquisition: Sensor captures and digitizes scene energy. Sets quality ceiling for everything downstream
  • Enhancement: Subjective improvement (contrast, brightness). No degradation model
  • Restoration: Objective recovery using mathematical degradation model (deblurring, denoising)
  • Morphological Processing: Shape-based analysis using structural elements; extracts boundaries, skeletons
  • Segmentation: Partitions image into meaningful regions. Hardest step: object variability, occlusion, ambiguous boundaries
  • Recognition: Assigns semantic labels to segmented objects

Enhancement vs Restoration: Enhancement is subjective (human judgment), restoration is objective (mathematical error). Enhancement has no model; restoration requires one.

6. Components of an Image Processing System

  • Image Sensors: Convert energy to electrical signals
  • Specialized Hardware: Dedicated processors for real-time pixel operations
  • Computer: Central processor running all algorithms
  • Software: Processing algorithms (filters to recognition)
  • Mass Storage: Large-capacity storage (1024×1024 8-bit image = 1 MB)
  • Displays: Monitors for visualization
  • Network/Cloud: Remote transmission and processing
  • Hardcopy: Printed output (films, prints)

7. Human Visual Perception

Eye structure: Cornea → Iris (light control) → Lens (focus) → Retina (sensor)

7.1 Cones vs Rods

7.2 Eye vs Camera Focusing

  • Camera: Fixed focal length; moves lens to focus
  • Eye: Fixed lens-retina distance (~17 mm); changes lens shape to focus (accommodation, focal length 14–17 mm)

8. Image Sensing and Acquisition

Three elements: Illumination source (emits energy) → Scene (reflects/absorbs/transmits) → Sensor (captures energy as electrical signal)

  • Single sensor: One element (photodiode); must scan to build image
  • Line sensor: Row of sensors; one line at a time (flatbed scanner)
  • Array sensor: 2D grid; captures full image at once (CCD/CMOS)

9. Image Formation Model

f(x, y) = i(x, y) × r(x, y)

  • i(x, y): Illumination; energy falling on scene. Range: 0 ≤ i < ∞
  • r(x, y): Reflectance; fraction of energy reflected. Range: 0 ≤ r ≤ 1 (0 = black, 1 = mirror)
  • Therefore: 0 ≤ f(x, y) < ∞
  • Different lighting on same object → different image (i changes)
  • Same lighting on different objects → different image (r changes)
  • X-ray special case: Uses transmissivity instead of reflectance (energy passes through)

10. Sampling and Quantization

10.1 Sampling (Space)

  • Places M × N grid over continuous image; records intensity at grid points only
  • Coordinates: x = 0 to M−1 (rows), y = 0 to N−1 (columns)
  • More samples = finer detail; fewer = loss of detail

10.2 Quantization (Intensity)

  • Maps continuous intensity to L = 2k discrete levels (k = bits/pixel)
  • 8-bit: 256 levels | 4-bit: 16 levels | 1-bit: 2 levels (binary)

10.3 Matrix Representation

  • Origin (0,0) at top-left; x goes downward, y goes rightward
  • Center: (floor(M/2), floor(N/2))
  • Storage: M × N × k bits. E.g., 512×512, 8-bit = 256 KB

11. Spatial and Intensity Resolution

AspectSpatial ResolutionIntensity Resolution
MeasuresSmallest discernible detailSmallest discernible intensity change
ParameterN (sampling rate / DPI)k (bits per pixel)
ArtifactJagged lines (aliasing)False contouring (posterization)
Sensitive toShape variationsShading variations

11.1 Iso-Preference Curves

  • Plot N vs k combinations giving same perceived quality
  • Shape-heavy images (crowd, text): need more pixels, tolerate fewer bits
  • Shade-heavy images (face, portrait): need more bits, tolerate fewer pixels

Set 2

Section A: 2 Marks Each
  1. What is spatial resolution?

    Ans: Smallest perceptible detail in an image. Measured in pixels per unit distance (DPI). Depends on sampling rate.

  2. What is intensity resolution?

    Ans: Smallest discernible change in intensity. Measured by bits per pixel (k). More bits = smoother gradations.

  3. Define reflectance in the image formation model.

    Ans: r(x, y) = fraction of illumination reflected by object. Range: 0 (total absorption) to 1 (total reflection). Depends on material properties.

  4. What are cones and rods?

    Ans: Cones (6–7M): fovea, color + detail, bright light (photopic). Rods (75–150M): peripheral, no color, dim light (scotopic).

  5. What is segmentation in DIP?

    Ans: Partitioning an image into meaningful regions/objects. Most challenging step in the DIP pipeline.

Section B: 2.5 Marks Each
  1. Explain f(x,y) = i(x,y) × r(x,y) with practical significance.

    Ans: f = illumination × reflectance. i depends on light source (0 to ∞), r depends on surface (0 to 1). Same object, different light = different image. Same light, different object = different image.

  2. What are iso-preference curves?

    Ans: Curves plotting N (samples) vs k (bits) combinations giving equal perceived quality. Shape-heavy images (crowd): tolerate fewer bits, need more pixels. Shade-heavy images (face): need more bits, tolerate fewer pixels.

  3. Describe the image acquisition process.

    Ans: Illumination source emits energy → scene reflects/absorbs/transmits → sensor (single/line/array) captures as electrical signal → sampled (M×N grid) and quantized (L = 2k levels) to produce digital image.

  4. What is morphological processing?

    Ans: Shape-based analysis using structural elements. Extracts boundaries, skeletons, connected components. Separates touching objects, fills holes, bridges pixel-level and object-level processing.

Section C: 5 Marks Each
  1. Compare spatial and intensity resolution with iso-preference curves.

    Ans:

    • Spatial: smallest detail, controlled by N, artifact = aliasing, sensitive to shapes
    • Intensity: smallest brightness change, controlled by k, artifact = false contouring, sensitive to shading
    • Both independent; can have high spatial with low intensity or vice versa
    • Iso-preference: shape-heavy images (crowd) tolerate fewer bits; shade-heavy (face) need more bits
    • Practical: text/crowd → more pixels; portraits → more bits
  2. Explain three levels of image processing with two examples each.

    Ans:

    • Low-level (Image → Image): noise reduction, contrast enhancement, sharpening
    • Mid-level (Image → Attributes): edge detection, segmentation, object description
    • High-level (Attributes → Understanding): scene recognition, autonomous navigation, medical diagnosis
    • Hierarchical: each level depends on quality of previous level's output

Set 3

Section A: 2 Marks Each
  1. What is false contouring?

    Ans: Artifact from low intensity resolution; smooth gradients appear as abrupt visible bands. Caused by too few bits (k) for quantization.

  2. What does L = 2k represent?

    Ans: L = number of intensity levels, k = bits per pixel. k=8 → 256 levels; k=1 → 2 levels (binary).

  3. What is the fovea?

    Ans: Central retina region densely packed with cones. Responsible for sharp, color vision in bright light. Direct gaze falls on fovea.

  4. Define image restoration.

    Ans: Objective recovery of degraded image using a mathematical model of the degradation. Success measured by mathematical error, not human judgment.

  5. Name four components of a DIP system.

    Ans: Image sensors, computer, mass storage, image display.

Section B: 2.5 Marks Each
  1. Explain sampling with the coordinate system.

    Ans: M×N grid over continuous image. x = 0 to M−1 (rows, downward), y = 0 to N−1 (columns, rightward). Origin (0,0) at top-left. Center at (floor(M/2), floor(N/2)). Higher M, N = finer detail.

  2. How does focusing differ between eye and camera?

    Ans: Camera: fixed focal length, moves lens to focus. Eye: fixed lens-retina distance (~17 mm), changes lens shape (accommodation, focal length 14–17 mm).

  3. Origins and evolution of DIP.

    Ans: 1920s: Bartlane cable, 5 gray levels, Atlantic image transmission. 1960s: space probe image enhancement, birth of modern DIP. 1970s: CT scanning, machines reveal what humans cannot perceive.

  4. Explain quantization with bit-depth example.

    Ans: L = 2k levels. k=8: 256 levels, smooth. k=4: 16 levels, slight banding. k=2: 4 levels, severe contouring. k=1: binary. More bits = smoother but more storage.

Section C: 5 Marks Each
  1. Image formation model with equation, constraints, significance, and X-ray case.

    Ans:

    • f(x, y) = i(x, y) × r(x, y)
    • i = illumination (0 ≤ i < ∞); r = reflectance (0 ≤ r ≤ 1, where 0 = black body, 1 = mirror)
    • Same object, different light = different image (i changes); vice versa for r
    • Homomorphic filtering separates i and r in log domain
    • X-ray: transmissivity replaces reflectance; dense materials (bone) transmit less, appear bright
  2. DIP pipeline with emphasis on why segmentation is hardest.

    Ans:

    • Pipeline: Acquisition → Enhancement/Restoration → Morphological → Segmentation → Recognition
    • Segmentation is hardest because: object variability (shape, size, color), occlusion (overlapping objects), ambiguous boundaries
    • Illumination effects create false edges or hide true ones
    • Errors cascade directly to recognition
    • Simple methods (thresholding) insufficient; real scenes need edge detection + region growing + clustering

Section A: 2 Marks Each
  1. What is photopic vision?

    Ans: Vision under bright light, mediated by cones. Provides color perception and high spatial detail.

  2. Define image enhancement.

    Ans: Subjective improvement of image appearance without a degradation model. Judged by “does it look better?” E.g., contrast stretching.

  3. What are spatial coordinates in an image?

    Ans: (x, y) position of each pixel. x = row (0 to M−1, downward), y = column (0 to N−1, rightward). Origin (0,0) at top-left.

  4. What is acquisition in DIP?

    Ans: First pipeline step: sensor captures scene energy, converts to digital signal via sampling and quantization. Sets quality ceiling.

  5. Two differences between rods and cones.

    Ans: (1) Cones: color, bright light (photopic); Rods: no color, dim light (scotopic). (2) Cones: fovea, 6–7M, high detail; Rods: peripheral, 75–150M, high sensitivity.

Section B: 2.5 Marks Each
  1. How is a digital image represented as a matrix?

    Ans: Matrix A of M×N where ai,j = f(i,j). Rows: 0 to M−1, columns: 0 to N−1. Origin (0,0) at top-left, center at (floor(M/2), floor(N/2)).

  2. What happens when spatial resolution is progressively reduced?

    Ans: Fine details blur first, then edges soften, text becomes unreadable, image becomes blocky/pixelated. Artifact: aliasing (jagged staircase edges).

  3. How does the EM spectrum serve as principal source of digital images?

    Ans: Each EM band reveals different information: gamma (nuclear), X-ray (bones), UV (fluorescence), visible (color), IR (heat), microwave (radar), radio (MRI). Same pipeline applies; only source and sensor differ.

  4. Differentiate enhancement and restoration with examples.

    Ans: Enhancement: subjective, no model (e.g., increasing contrast of foggy photo). Restoration: objective, uses degradation model (e.g., deconvolution to reverse motion blur). Success: human judgment vs mathematical error.

Section C: 5 Marks Each
  1. Sampling, quantization, and resolution tradeoffs for different image types.

    Ans:

    • Sampling: M×N grid; more samples = finer detail; too few = aliasing
    • Quantization: L = 2k levels; more bits = smoother; too few = false contouring
    • Storage = M × N × k bits
    • Shape-heavy (text, crowd): prioritize pixels (large N)
    • Shade-heavy (portrait, medical): prioritize bits (large k)
  2. Describe all components of an image processing system.

    Ans:

    • Sensors: Convert energy to electrical signals (single/line/array)
    • Specialized hardware: Dedicated processors (GPU/FPGA) for real-time ops
    • Computer: Central hub running all algorithms
    • Software: Algorithms from filters to recognition pipelines
    • Storage, Display, Network, Hardcopy: Archive, visualize, transmit, print

Set 6

Section A: 2 Marks Each
  1. Define illumination i(x, y).

    Ans: Energy from source falling on scene at point (x, y). Depends on light source only. Range: 0 to ∞.

  2. What is transmissivity?

    Ans: Fraction of energy passing through an object. Used instead of reflectance in X-ray imaging. Dense materials (bone) transmit less.

  3. Where is origin (0,0) and axis directions?

    Ans: Origin at top-left. x-axis (rows) goes downward (0 to M−1). y-axis (columns) goes rightward (0 to N−1).

  4. What is an iso-preference curve?

    Ans: Curve on N-k graph connecting combinations producing same perceived quality. Different image types have different curves.

  5. How did 1970s medical imaging advance DIP?

    Ans: CT scanning reconstructed body structures from X-ray projections. Proved machines can reveal what humans cannot perceive.

Section B: 2.5 Marks Each
  1. Why is segmentation the hardest step?

    Ans: Object variability (shape, size, color), occlusion (overlapping), ambiguous edges, illumination effects (false edges), cascading errors to recognition.

  2. How is a digital image stored as a matrix?

    Ans: M×N matrix, ai,j = f(i,j). Rows 0 to M−1, columns 0 to N−1. Each element: k bits. Storage = M×N×k bits. E.g., 512×512×8 = 256 KB.

  3. Ranges of reflectance and illumination. Why is r bounded by 1?

    Ans: i: 0 to ∞ (no upper limit on source energy). r: 0 to 1 (fraction of light reflected). Cannot reflect more than received, hence upper bound 1. Black velvet ~0.01, snow ~0.93.

  4. Sensitivity of spatial and intensity resolution to image content.

    Ans: Spatial: sensitive to shapes; fine geometry (text, edges) degrades fast with fewer pixels. Intensity: sensitive to shading; smooth gradients (skin, sky) degrade fast with fewer bits.

Section C: 5 Marks Each
  1. Journey from physical scene to stored digital image.

    Ans:

    • Formation: f(x,y) = i(x,y) × r(x,y). i: 0 to ∞, r: 0 to 1
    • Sensing: Sensor (single/line/array) converts reflected energy to continuous electrical signal
    • Sampling: M×N grid; origin top-left; too few = aliasing
    • Quantization: L = 2k levels; too few = false contouring
    • Storage: M×N matrix, total = M×N×k bits
  2. DIP pipeline with methods at each stage.

    Ans:

    • Acquisition: CCD/CMOS, X-ray detectors, radar antennas
    • Enhancement: Contrast stretching, histogram equalization, spatial filtering

    • Recognition: Template matching, feature-based classification, neural networks
