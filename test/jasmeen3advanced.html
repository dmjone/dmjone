Part 1: Advanced Conditional Logic

Complex Conditional Structures

Complex conditional structures are used to evaluate multiple conditions in decision-making processes. They allow a program to perform different actions based on the evaluation of multiple logical expressions. Logical operators such as AND, OR, and NOT are used to combine conditions.

Syntax:


if condition1 and condition2:
    # Executes if both conditions are true
elif condition1 or condition2:
    # Executes if at least one condition is true
else:
    # Executes if none of the above conditions are true
    
# Example of complex conditional structures
x = 15
y = 20

# Check if both conditions are met
if x > 10 and y < 25:
    print("Both conditions are met.")  # This will be printed because x > 10 and y < 25 are both true

# Check if at least one condition is met
elif x > 10 or y == 20:
    print("At least one condition is met.")  # This block is not executed because the previous condition was true

# If none of the above conditions are met
else:
    print("No conditions are met.")  # This block is not executed because one of the above conditions was true

Conditional Ternary Operator

The ternary operator provides a concise way to perform conditional evaluations in a single line of code. It is useful for simple conditional assignments and makes the code more readable and compact.

Syntax:


# Syntax of the ternary operator
variable = value_if_true if condition else value_if_false
    
# Ternary operator for quick conditional evaluation
temperature = 25

# Using ternary operator to determine status based on temperature
status = "Warm" if temperature >= 20 else "Cold"

# Print the result
print("It is", status)  # This will print: It is Warm because temperature is 25 which is >= 20

Switch Case Statements

Switch case statements provide a cleaner alternative to multiple if-else statements by allowing a variable to be tested for equality against a list of values. In languages that do not natively support switch case, like Python, similar functionality can be achieved using dictionary mappings.

Syntax:


# In Python, switch-case can be simulated using dictionary mapping
def switch_case(variable):
    switcher = {
        case1: result1,
        case2: result2,
        ...
        caseN: resultN
    }
    return switcher.get(variable, default_result)
    
# Simulating switch-case using dictionary mapping in Python

# Function to simulate switch-case
def switch_case(day):
    # Dictionary mapping for days of the week
    switcher = {
        1: "Monday",
        2: "Tuesday",
        3: "Wednesday",
        4: "Thursday",
        5: "Friday",
        6: "Saturday",
        7: "Sunday"
    }
    # Get the day name from the switcher dictionary, default to "Invalid day"
    return switcher.get(day, "Invalid day")

# Print the result for day 4
print(switch_case(4))  # This will print: Thursday because 4 maps to "Thursday" in the switcher dictionary

Part 2: Mastery of Loops

Advanced For Loops

Advanced for loops include using multiple variables, nested loops, and unconventional iterations to handle complex data structures and algorithms. These techniques provide greater control and flexibility in looping constructs.

Why Nested For Loops?

Nested for loops are required when dealing with multi-dimensional data structures like matrices, tables, or grids. They allow iteration over multiple dimensions, enabling operations such as matrix multiplication, table traversal, or generating combinations of elements.

When to Use Nested For Loops?

We use nested for loops when a problem requires iteration over multiple levels of data or when an operation depends on combinations of elements from different sequences. If a problem involves processing elements in a grid or comparing pairs of elements, nested loops provide a structured and efficient solution compared to multiple single-level loops.

Syntax:


# For loop with multiple variables
for var1, var2 in zip(sequence1, sequence2):
    # Code block to execute

# Nested for loop
for var1 in sequence1:
    for var2 in sequence2:
        # Code block to execute
    
# Using a for loop with multiple variables

# Loop through two ranges simultaneously using zip
for i, j in zip(range(5), range(5, 10)):
    # Print values of i and j
    print("i:", i, "j:", j)

# Example of nested for loop

# Outer loop runs 3 times
for i in range(3):
    # Inner loop runs 3 times for each iteration of the outer loop
    for j in range(1, 4):
        # Print the product of i and j
        print(f"{i} * {j} = {i*j}")

While Loops and Efficiency

Best practices for using while loops effectively, avoiding common pitfalls like infinite loops, and optimizing loop performance. While loops are ideal for scenarios where the number of iterations is not predetermined, and they can offer more flexibility than for loops.

Best Practices:

  • Ensure the loop has a clear exit condition to avoid infinite loops.
  • Update control variables within the loop to progress towards the exit condition.
  • Optimize loop performance by minimizing the code inside the loop.

Syntax:


# Basic while loop syntax
while condition:
    # Code block to execute
    # Ensure control variable is updated to avoid infinite loops
    
# Efficient use of while loop with a control variable

# Initialize control variable
i = 0

# While loop with exit condition
while i < 10:
    # Print current value of i
    print(i)
    
    # Increment control variable to avoid infinite loop
    i += 1

Using Break, Continue, and Pass

Strategic use of control statements within loops to manage flow more effectively. These statements help in controlling the loop execution based on specific conditions, enhancing code readability and efficiency.

Syntax:


# Break statement: exits the loop immediately
for item in iterable:
    if condition:
        break

# Continue statement: skips the current iteration and proceeds to the next
for item in iterable:
    if condition:
        continue

# Pass statement: does nothing, acts as a placeholder
for item in iterable:
    pass
    
# Using break to exit loop

# Loop through numbers 0 to 9
for num in range(10):
    # Exit loop when num equals 6
    if num == 6:
        break
    # Print the current number
    print(num)

# Using continue to skip current iteration

# Loop through numbers 0 to 9
for num in range(10):
    # Skip the iteration if num is even
    if num % 2 == 0:
        continue
    # Print the current number (only odd numbers will be printed)
    print(num)

# Using pass as a placeholder

# Loop through numbers 0 to 4
for num in range(5):
    pass  # Placeholder to do nothing in this loop

Loop Unrolling and Other Optimization Techniques

Introduction to techniques like loop unrolling to enhance performance in critical sections of code. Loop unrolling reduces the overhead of loop control, leading to performance improvements, especially in performance-critical applications.

Why Use Loop Unrolling?

Loop unrolling is used to minimize the overhead of loop control and increase the performance of loops by reducing the number of iterations. This technique is beneficial in scenarios where the loop body is small, and the loop control overhead is significant.

Syntax:


# Standard loop
for i in range(n):
    # Code block

# Unrolled loop
for i in range(0, n, unroll_factor):
    # Code block replicated unroll_factor times
    
# Loop unrolling example

# Standard loop
for i in range(10):
    print(i)

# Unrolled loop with unroll factor of 2
for i in range(0, 10, 2):
    print(i)        # Print current index
    print(i + 1)    # Print next index (unrolled iteration)

Assignment 2: Batch Processing with Nested Loops

Implement a program to process data in batches using nested loops and complex conditional logic.

Task:

Given a large list of numerical data, process the data in batches. For each batch, perform the following operations:

  • Calculate the sum of the batch.
  • If the sum is above a certain threshold, mark the batch as 'High'.
  • If the sum is below the threshold, mark the batch as 'Low'.
  • Store the results with batch identifiers.

# Sample data
data = [5, 15, 25, 35, 45, 55, 65, 75, 85, 95]

# Batch size and threshold value
batch_size = 3
threshold = 100

# List to store batch processing results
batch_results = []

# Process data in batches using nested loops
for i in range(0, len(data), batch_size):
    batch = data[i:i + batch_size]
    batch_sum = sum(batch)
    
    if batch_sum > threshold:
        batch_results.append({'batch_id': i // batch_size, 'sum': batch_sum, 'status': 'High'})
    else:
        batch_results.append({'batch_id': i // batch_size, 'sum': batch_sum, 'status': 'Low'})

# Print the batch results
for result in batch_results:
    print(result)
