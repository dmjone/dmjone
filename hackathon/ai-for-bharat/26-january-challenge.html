Setu is an AI-powered bridge that connects rural farmers to the ONDC network via voice commands in local dialects. Using Google Gemini, it translates spoken intent into structured Beckn protocol messages, democratizing digital commerce access.

Deployment Status: The application is live at aib26.dmj.one.
Note: This instance is hosted on a Google Cloud Spot VM to optimize costs. As Spot VMs can be preempted by Google at any time, if the link is inaccessible, it indicates the instance has been terminated.
