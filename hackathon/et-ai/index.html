4 Satellite Sources via Earth Engine

Sentinel-2 (ESA)

10m resolution

NDVI, EVI, NDWI vegetation indices. 30-day cloud-free composite. Crop health: Healthy / Moderate / Stressed. NDVI trajectory with district benchmark.

Sentinel-1 SAR (ESA)

10m · Works through clouds

C-band VV/VH radar backscatter. Soil moisture classification (wet/moist/dry/very_dry). Vegetation density. Works when optical imagery is blocked by clouds.

MODIS Terra (NASA)

1km · Daily

Land Surface Temperature (day/night). Heat stress detection (none/moderate/high/extreme). Regional anomaly comparison. Diurnal range for fruit setting.

NASA SMAP L4

9km · Root-zone 0–100cm

Surface moisture + root-zone moisture. Detects "surface wet but roots dry" situations. Critical for irrigation timing. 3-hourly updates.

Market + Weather + Location

AgMarkNet (data.gov.in)

112 commodities

Live mandi prices + 90-day price history. Net profit ranking (price − transport − commission − spoilage). Price-weather correlation analysis.

Open-Meteo

Hyperlocal

5-day hourly forecast + 90-day historical. Growing Degree Days (GDD) for crop stage estimation. 10 crop growth models.

Google Maps Platform

Distance Matrix + Places

Real driving distances to every mandi. Transport cost at ₹3.5/km/quintal. Nearest KVK (Krishi Vigyan Kendra) search.

Voice Pipeline

22 languages

Cloud STT V2 (speech recognition) + Cloud TTS Wavenet (10 Indian voices) + Cloud Translation v3. Gemini Live for WebSocket voice streaming.

Cross-Validation Engine

Detects contradictions across data sources to prevent misleading advice:

Conflict Sources Action
NDVI declining + adequate rain Sentinel-2 vs Open-Meteo Flag pest/disease → refer KVK (NOT irrigation)
NDVI declining + SAR confirms dry Sentinel-2 vs Sentinel-1 High-confidence irrigation recommendation
Heat stress + flowering stage MODIS vs GDD model Crop protection alert (shade/irrigation)
Rain forecast + harvest-ready Open-Meteo vs GDD model Urgent harvest-before-rain warning
Surface wet but root-zone dry SMAP surface vs root-zone Deep irrigation needed — rain didn't reach roots
Price rising + high arrivals AgMarkNet trend analysis Hedge: sell partial now, hold rest

Error Handling & Resilience

  • 5-model fallback — Gemini 3 Flash → 2.5 Flash → 2.0 Flash → 2.0 Flash Lite → 1.5 Flash + Vertex AI
  • Retry on 429 — Extracts API retry delay, waits (capped 15s), retries once per model
  • 3-tier cache — L0 satellite grid (O(1), 3,788 points) + L1 in-memory (0.13s) + L2 Cloud Storage (~200ms)
  • Graceful degradation — If one satellite fails, others still return; advisory adapts to available data
  • Confidence gating — LOW data omitted, MEDIUM hedged, only HIGH stated as advice
  • Session cleanup — Auto-evicts stale sessions (>1hr) to prevent memory leaks
  • Fact-check — Gemini verifies advisory against source data before delivery

Worked Example: Solan Tomato Farmer

Without KisanMind: Sells at local Solan mandi at ₹1,800/quintal. Loses ₹10,000/year to rain-damaged harvest. Doesn't know root-zone is drying (SMAP data unavailable). Income: ₹50,000/year.

With KisanMind: Sentinel-2 NDVI 0.54 (moderate). Sentinel-1 SAR: moist soil — rules out water stress. SMAP: root-zone adequate. Cross-validation flags: "NDVI declining despite moisture → possible pest → refer KVK." Weather warns rain in 72 hours → harvest today. Shimla mandi ₹2,400/q (60km); net ₹2,104/q after transport. 90-day history: above average → sell now.

Result: ₹376/q × 30q = ₹11,280 gained + ₹10,000 spoilage saved = ₹21,280 per harvest.
