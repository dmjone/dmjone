dmj.one | Code of Honor

To build trust. To grow together.

1. Speak Truth, Not Behind Backs If there’s an issue, talk to the person–not about them.
2. Give Credit, Share Blame We grow by lifting others–not by stepping on them.
3. Ask Before You Share Private messages stay private unless given permission.
4. No Revenge, No Games Disagree? Speak. Don’t plot. We solve–not sabotage.
5. Help with Heart, Not for Power Support is sacred. Never weaponize someone’s help.
6. Stay Accountable If you made a mistake, own it. If they did, confront respectfully.
7. Loyalty is Earned, Not Assumed Trust takes time. Don’t fake it. Don’t break it.
We rise—not to be above, but to lift each other higher.
